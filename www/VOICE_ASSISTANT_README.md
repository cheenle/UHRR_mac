# 语音助手功能说明

## 概述

语音助手是MRRC系统的智能语音识别与合成功能，支持：
- **实时语音识别**：将电台接收的SSB语音转换为文字
- **语音合成发射**：将输入的文字合成为语音并通过电台发射
- **对话记录**：保存收发对话历史
- **快捷回复**：预设常用业余无线电通联用语

## 访问方式

1. 打开移动端主界面 `mobile_modern.html`
2. 点击左上角菜单按钮 (☰)
3. 选择 **"🎙️ 语音助手"**

## 功能说明

### 1. 语音识别显示区

- **实时转写**：自动识别电台接收的语音并显示文字
- **置信度指示**：显示识别可信度（绿色=高，黄色=中，红色=低）
- **音频可视化**：实时显示接收音频波形

### 2. 对话记录区

- 显示所有收发的消息历史
- 接收的消息显示在左侧（灰色气泡）
- 发送的消息显示在右侧（蓝色气泡）
- 支持清空历史记录

### 3. 回复输入区

- **文字输入**：手动输入要发送的回复内容
- **语音输入**：点击"语音输入"按钮，直接语音输入回复
- **快捷回复**：点击"快捷回复"按钮，选择预设常用语

### 4. 语音控制

- **试听按钮**：先试听合成的语音效果
- **PTT发射按钮**：
  - **按住**：发射语音（同时播放合成语音）
  - **松开**：停止发射
  - 红色大按钮，带波纹动画效果
- **停止按钮**：立即停止语音播放和发射

## 设置选项

点击右上角设置按钮 (⚙️) 可配置：

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| 本地语音识别 | 使用浏览器内置识别API | 开启 |
| 本地语音合成 | 使用浏览器内置TTS | 开启 |
| 自动PTT发射 | 合成完成后自动发射 | 关闭 |
| 语音识别语言 | 中文/English/日本語 | 中文 |
| 发射前延迟 | 合成完成后等待时间 | 1秒 |

## 快捷回复预设

系统预设了常用业余无线电通联用语：

1. **📡 标准回复** - "这里是BH4XXX，信号报告59，您那边情况如何？"
2. **👍 信号良好** - "信号很好，清晰可辨，继续"
3. **🔁 请求重复** - "信号较弱，请重复一遍"
4. **❓ 询问呼号** - "这里是BH4XXX，QRZ？"
5. **👋 结束通联** - "73，再见！"
6. **📍 询问位置** - "请报一下您的位置"

## 技术实现

### 语音识别
- **Web Speech API**：使用浏览器内置的语音识别API
- **支持语言**：中文、英文、日文
- **实时性**：流式识别，边说边出文字

### 语音合成
- **Web Speech Synthesis**：浏览器内置TTS
- **语音选择**：自动选择中文语音
- **语速调节**：支持慢速/正常/快速

### PTT集成
- 与主界面的PTT控制无缝集成
- 支持长按发射，松开停止
- 自动处理音频路由

## 注意事项

1. **麦克风权限**：首次使用需要允许麦克风访问权限
2. **网络连接**：语音识别需要网络连接（浏览器API要求）
3. **安静环境**：在安静环境下使用识别准确率更高
4. **语速适中**：正常语速识别效果最佳

## 故障排除

### 语音识别不工作
1. 检查麦克风权限是否开启
2. 检查网络连接是否正常
3. 尝试刷新页面重新授权

### 语音合成不播放
1. 检查设备音量是否开启
2. 尝试选择不同的语音
3. 检查浏览器是否支持TTS

### PTT发射失败
1. 确保主界面已正常连接电台
2. 检查电源按钮是否已开启
3. 确认WebSocket连接正常
