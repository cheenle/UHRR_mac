安装指南

快速开始

# 克隆仓库
git clone https://github.com/cheenle/UHRR.git
cd UHRR

# 安装依赖
pip3 install -r requirements.txt

# 启动服务
./mrrc_control.sh start

系统要求

  • Python 3.12+
  • macOS 或 Linux
  • Hamlib/rigctld
  • TLS证书（可选）

详细步骤

1. 环境准备

确保系统已安装Python 3.12或更高版本

python3 --version  # 应显示 3.12.x

2. 安装依赖

pip3 install numpy tornado pyaudio sounddevice

3. 配置电台

编辑 MRRC.conf 配置文件

[HAMLIB]
 rig_model = IC_M710
 rig_pathname = /dev/cu.usbserial-230
 rig_rate = 4800
