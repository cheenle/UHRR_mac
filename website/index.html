Core Features

Built for Modern HAM Radio

Mobile First

Touch-optimized interface, PWA support for home screen, perfect for iPhone/Android.

Ultra Low Latency

TX/RX switch <100ms, Opus codec optimization, AudioWorklet playback, professional real-time experience.

WDSP Professional NR

Integrated OpenHPSDR WDSP library, NR2 spectral NR 15-20dB, full AGC/ANF/NB support.

AI Voice Assistant

Whisper speech recognition + Qwen3-TTS synthesis, voice control, automatic phonetic callsigns.

CW Real-time Decode

Frontend ONNX inference (<2MB), dual-mode architecture, QSO state machine smart reply suggestions.

ATR-1000 Integration

Real-time power/SWR display, smart tuner learning, auto-tune parameter storage and quick recall.

Multi-Instance Support

Run multiple independent instances on a single server, independent radio control, differentiated tuner learning.

FT8 Integration

ULTRON automation tool, JTDX/WSJT-X linkage support, DXCC target tracking, smart QSO.

NanoVNA Integration

Built-in vector network analyzer Web interface, PWA offline support, antenna parameter analysis.
