# 2. Business Direction (BUS 411)

## 2.1 Vision

Become the leading mobile-first amateur radio remote control solution, enabling HAM operators to enjoy radio communication anytime, anywhere through modern web technologies.

## 2.2 Mission

Through innovative web technology and professional DSP processing, provide amateur radio operators with low-latency, high-reliability remote radio control experience accessible from any modern browser — no native app installation required.

## 2.3 Goals

| ID | Goal | Description |
|----|------|-------------|
| G1 | Mobile Leadership | Become the preferred mobile HAM radio remote control solution |
| G2 | Latency Excellence | Achieve industry-leading TX/RX switching delay (< 100ms) |
| G3 | PTT Reliability | Maintain 99%+ push-to-talk reliability |
| G4 | DSP Integration | Integrate professional-grade WDSP noise reduction (15-20dB) |

## 2.4 Objectives

| ID | Objective | Metric | Target |
|----|-----------|--------|--------|
| O1 | TX/RX Switching Delay | Milliseconds | < 100ms |
| O2 | PTT Reliability | Success rate | >= 99% |
| O3 | Noise Reduction Depth | Decibels | >= 15dB |
| O4 | Mobile User Satisfaction | Rating scale 1-5 | >= 4.5 |
| O5 | System Availability | Uptime percentage | >= 99.5% |

## 2.5 Strategies

| ID | Strategy | Description |
|----|----------|-------------|
| S1 | Mobile-First Design | All features designed for mobile experience first, desktop second |
| S2 | Performance Optimization | Continuously optimize latency and reliability |
| S3 | DSP Deep Integration | Integrate professional WDSP library for signal quality |
| S4 | Open Source Ecosystem | Build on open source, contribute back to community |

## 2.6 Tactics

| ID | Tactic | Description |
|----|--------|-------------|
| T1 | AudioWorklet | Low-latency browser audio playback |
| T2 | PTT Warm-up Frames | Pre-transmit audio frames to eliminate switching gaps |
| T3 | WDSP NR2 | Spectral noise reduction for SSB optimization |
| T4 | Multi-instance Deployment | Support multiple independent radio instances |
| T5 | Opus Codec | Low-bandwidth, high-quality audio compression |
