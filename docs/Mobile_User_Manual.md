## 1. 快速开始

### 1.1 访问系统

在移动设备浏览器中访问：
```
https://<您的服务器地址>/mobile_modern.html
```

> **推荐浏览器**：Safari (iOS)、Chrome (Android)

### 1.2 登录认证

首次访问会要求输入用户名和密码。请联系系统管理员获取登录凭据。

### 1.3 授予权限

系统需要以下权限：
- **麦克风**：用于发射音频
- **通知**（可选）：用于状态提醒

## 3. 核心功能

### 3.1 开关机

**开启系统：**
1. 点击左上角电源按钮
2. 等待连接建立（按钮变为绿色）
3. 频率显示将更新为电台当前频率

**关闭系统：**
1. 再次点击电源按钮
2. 连接断开（按钮变为红色）

### 3.2 频率调节

**方式一：按钮调节**
- 点击 `+` 按钮增加频率
- 点击 `-` 按钮减少频率
- 可选择步进值：+5kHz / +50kHz

**方式二：波段选择**

![波段选择](../screenshots/band_select.png)

通过菜单 → Band Selection 快速切换常用业余频段：
- 160m (1.8-2.0 MHz)
- 80m (3.5-4.0 MHz)
- 40m (7.0-7.3 MHz) ⭐ 推荐
- 20m (14.0-14.35 MHz)
- 15m (21.0-21.45 MHz)
- 10m (28.0-29.7 MHz)

### 3.3 模式切换

![模式选择](../screenshots/mode_selection.png)

通过菜单 → Mode Selection 切换工作模式：

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| **USB** | 上边带 | 20m及以上波段 |
| **LSB** | 下边带 | 40m及以下波段 |
| **AM** | 调幅 | 调幅广播接收 |
| **FM** | 调频 | VHF/UHF FM通信 |

> **提示**：当前选中模式会以绿色高亮显示。

### 3.4 发射（PTT）

**发射操作：**
1. 确保电源已开启（绿色状态）
2. 按住底部 PTT 按钮
3. 对着麦克风说话
4. 松开 PTT 按钮结束发射

> **注意**：PTT 按钮按下时会触发震动反馈，表示发射开始。

**TUNE 天调模式：**
1. 长按 TUNE 按钮
2. 系统将发射 1kHz 单音
3. 天调自动匹配天线
4. 松开按钮停止发射

## 4. 高级功能

### 4.1 功率与SWR监测

当连接 ATR-1000 设备时，主界面会实时显示：

| 参数 | 范围 | 说明 |
|------|------|------|
| 功率 | 0-200W | 前向功率 |
| SWR | 1.0-9.99 | 驻波比 |

> **提示**：功率显示仅在发射时更新，延迟 <200ms。理想 SWR 值为 1.0。

### 4.2 TX 均衡器

![TX 均衡器](../screenshots/EQ_TX.png)

优化发射音频质量，位于菜单 → TX Equalizer。

**预设模式：**

| 模式 | 参数 (低/中/高) | 适用场景 |
|------|-----------------|----------|
| 默认 | 0 / 0 / 0 dB | 无处理 |
| 短波语音 | +4 / +6 / -3 dB | 常规短波通信 |
| 弱信号 | +6 / +8 / -6 dB | DX 弱信号通信 |
| 比赛模式 | +2 / +4 / -2 dB | 快速通联 |

> **说明**：短波通信中高频信号容易过强导致声音刺耳，选择合适预设可增强中低频，提高语音清晰度。

### 4.3 音频设置

![音频设置](../screenshots/Audio_volum.png)

通过菜单 → Audio Controls 调节：

| 参数 | 范围 | 说明 |
|------|------|------|
| AF 增益 | 0-100% | 接收音量 |
| MIC 增益 | 0-100% | 发射音量 |
| 静噪 | 0+ | 静噪门限 |

**调节建议：**
- AF 增益：根据耳机/扬声器调节，避免失真
- MIC 增益：50% 为起点，根据对方反馈调整
- 静噪：弱信号时设为 0，强信号时可适当提高

### 4.4 PWA 离线使用

**添加到主屏幕：**

**iOS Safari：**
1. 点击分享按钮
2. 选择 "添加到主屏幕"
3. 命名并确认

**Android Chrome：**
1. 点击菜单按钮
2. 选择 "添加到主屏幕"
3. 确认添加

> 添加后可离线访问基本界面，但实时功能需要网络连接。

## 5. 设置与配置

### 5.1 连接状态指示

| 指示器 | 状态 | 说明 |
|--------|------|------|
| 电源按钮 | 🟢 绿色 | 系统已连接 |
| 电源按钮 | 🔴 红色 | 系统未连接 |
| PTT按钮 | 🟢 绿色 | 正在发射 |
| PTT按钮 | ⚪ 白色 | 待命状态 |
| S表 | 动态条 | 信号强度 (S0-S9+60dB) |

### 5.2 WDSP 数字处理设置

从菜单 → Settings → 🎛️ WDSP 数字处理

| 设置项 | 说明 | 推荐值 |
|--------|------|--------|
| WDSP 处理 | 主开关 | 开启 |
| NR2 降噪 | 频谱降噪级别 (0-4) | 2-3 |
| NB 抑制 | 脉冲噪声抑制 | 根据环境 |
| ANF 陷波 | 自动消除CW干扰 | 有CW干扰时开启 |
| AGC 模式 | 自动增益控制 | SLOW 或 MED |

**WDSP 效果**：NR2频谱降噪可让SSB语音听起来像FM，显著降低背景噪声。

### 5.3 性能指标

| 指标 | 数值 |
|------|------|
| TX 延迟 | ~65ms |
| RX 延迟 | ~51ms |
| TX→RX 切换 | <100ms |
| PTT 可靠性 | 99%+ |
| 功率显示延迟 | <200ms |
| WDSP 降噪 | 15-20dB |

## 6. 常见问题

### Q1: PTT 按钮不响应？

**解决方案：**
1. 确认电源已开启（绿色状态）
2. 检查麦克风权限是否已授权
3. 刷新页面重新连接
4. 检查浏览器控制台是否有错误

### Q2: 频率显示不正确？

**解决方案：**
1. 刷新页面重新获取频率
2. 检查后端 rigctld 是否正常运行
3. 确认电台已开机

### Q3: 功率/SWR 不显示？

**解决方案：**
1. 确认 ATR-1000 代理是否运行
2. 检查 ATR-1000 设备是否在线
3. 查看后端日志确认连接状态

### Q4: 音频有噪音或断续？

**解决方案：**
1. 检查网络连接质量
2. 尝试调节 AF/MIC 增益
3. 关闭其他占用带宽的应用

### Q5: 如何查看连接状态？

**解决方案：**
1. 查看电源按钮颜色
   - 绿色：已连接
   - 红色：未连接
2. 打开浏览器控制台查看详细日志
