## 1. 快速开始

### 1.1 访问系统

在移动设备浏览器中访问：
```
https://<您的服务器地址>/mobile_modern.html
```

> **推荐浏览器**：Safari (iOS)、Chrome (Android)

### 1.2 登录认证

首次访问会要求输入用户名和密码。请联系系统管理员获取登录凭据。

### 1.3 授予权限

系统需要以下权限：
- **麦克风**：用于发射音频
- **通知**（可选）：用于状态提醒

## 3. 核心功能

### 3.1 开关机

**开启系统：**
1. 点击左上角电源按钮
2. 等待连接建立（按钮变为绿色）
3. 频率显示将更新为电台当前频率

**关闭系统：**
1. 再次点击电源按钮
2. 连接断开（按钮变为红色）

### 3.2 频率调节

**方式一：按钮调节**
- 点击 `+` 按钮增加频率
- 点击 `-` 按钮减少频率
- 长按可连续调节

**方式二：直接输入**
1. 点击频率显示区域
2. 在弹出键盘输入频率
3. 点击确认

**方式三：步进选择**
- 可选择不同步进值：10kHz / 5kHz / 1kHz

### 3.3 模式切换

通过设置菜单切换工作模式：
- **LSB**：下边带（适用于 40m 及以下波段）
- **USB**：上边带（适用于 20m 及以上波段）
- **AM**：调幅模式
- **CW**：电报模式

### 3.4 发射（PTT）

**发射操作：**
1. 确保电源已开启（绿色状态）
2. 按住底部 PTT 按钮
3. 对着麦克风说话
4. 松开 PTT 按钮结束发射

> **注意**：PTT 按钮按下时会触发震动反馈，表示发射开始。

**TUNE 天调模式：**
1. 长按 TUNE 按钮
2. 系统将发射 1kHz 单音
3. 天调自动匹配天线
4. 松开按钮停止发射

## 4. 高级功能

### 4.1 功率与SWR监测

当连接 ATR-1000 设备时，界面会实时显示：

| 参数 | 说明 |
|------|------|
| 功率 | 前向功率（0-200W） |
| SWR | 驻波比（1.0-9.99） |

> **提示**：功率显示仅在发射时更新，延迟 <200ms。

### 4.2 TX 均衡器

优化发射音频质量：

**预设模式：**
| 模式 | 适用场景 |
|------|----------|
| 默认 | 无处理 |
| 短波语音 | 常规短波通信 |
| 弱信号 | DX 弱信号通信 |
| 比赛模式 | 快速通联 |

**调节方式：**
1. 打开设置菜单
2. 选择 "TX EQ"
3. 选择预设模式
4. 设置自动保存

### 4.3 天调存储

系统自动记录天调参数：
- 每次天调成功后自动保存
- 切换频率时自动加载匹配参数
- 频率匹配容差：±50kHz

### 4.4 音量控制

**AF 增益：**
1. 主界面滑动音量滑块
2. 范围：0-100%
3. 实时调节接收音量

**MIC 增益：**
1. 通过设置菜单调节
2. 范围：0-100%
3. 影响发射音频电平

### 4.5 PWA 离线使用

**添加到主屏幕：**

**iOS Safari：**
1. 点击分享按钮
2. 选择 "添加到主屏幕"
3. 命名并确认

**Android Chrome：**
1. 点击菜单按钮
2. 选择 "添加到主屏幕"
3. 确认添加

> 添加后可离线访问基本界面，但实时功能需要网络连接。

## 5. 设置与配置

### 5.1 设置菜单

点击右上角设置图标进入设置菜单：

**音频设置：**
- AF 增益：接收音量
- MIC 增益：发射音量
- 静噪：静噪电平

**显示设置：**
- 频率格式：MHz / kHz
- 主题：自动 / 浅色 / 深色

**高级设置：**
- TX EQ：发射均衡器
- 滤波器：音频滤波选项

### 5.2 波段选择

通过菜单快速切换常用业余频段：
- 160m (1.8-2.0 MHz)
- 80m (3.5-4.0 MHz)
- 40m (7.0-7.3 MHz) ⭐ 推荐
- 20m (14.0-14.35 MHz)
- 15m (21.0-21.45 MHz)
- 10m (28.0-29.7 MHz)

## 6. 常见问题

### Q1: PTT 按钮不响应？

**解决方案：**
1. 确认电源已开启（绿色状态）
2. 检查麦克风权限是否已授权
3. 刷新页面重新连接
4. 检查浏览器控制台是否有错误

### Q2: 频率显示不正确？

**解决方案：**
1. 刷新页面重新获取频率
2. 检查后端 rigctld 是否正常运行
3. 确认电台已开机

### Q3: 功率/SWR 不显示？

**解决方案：**
1. 确认 ATR-1000 代理是否运行
2. 检查 ATR-1000 设备是否在线
3. 查看后端日志确认连接状态

### Q4: 音频有噪音或断续？

**解决方案：**
1. 检查网络连接质量
2. 尝试降低音频质量设置
3. 关闭其他占用带宽的应用

### Q5: 如何查看连接状态？

**解决方案：**
1. 查看电源按钮颜色
   - 绿色：已连接
   - 红色：未连接
2. 打开浏览器控制台查看详细日志
