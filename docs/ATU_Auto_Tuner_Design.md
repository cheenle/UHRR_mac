# ATU 自动调谐功能需求说明

## 1. 功能概述

### 1.1 目标
为 MRRC 系统添加 ATU (Antenna Tuning Unit) 自动调谐功能，支持对业余无线电短波波段进行全自动频率扫描和天调参数优化，并将结果存储到映射表中。

### 1.2 核心价值
- **一键覆盖**：选择波段后自动扫描整个频段，无需手动逐点调谐
- **智能优化**：基于已有数据智能选择初始参数，微调 C/L 找到最佳值
- **可视化**：实时显示频率-SWR 曲线图，直观了解天线特性
- **持久化**：所有调谐结果自动保存，下次使用直接调用

## 2. 用户场景

### 2.1 场景一：新天线调谐
用户安装了新天线，需要对整个 40m 波段进行调谐：
1. 打开 ATU 调谐界面
2. 选择 "40m 波段" (7.0-7.2 MHz)
3. 点击 "开始自动调谐"
4. 系统自动扫描 200 个频点（1kHz 步进）
5. 查看频率-SWR 曲线图
6. 所有参数已保存，后续使用直接调用

### 2.2 场景二：特定频率范围调谐
用户需要在 14.200-14.350 MHz 范围内进行调谐：
1. 打开 ATU 调谐界面
2. 选择 "自定义范围"
3. 输入起始频率 14200 kHz，结束频率 14350 kHz
4. 点击 "开始自动调谐"
5. 系统扫描 151 个频点
6. 查看 SWR 曲线，找到最佳工作频率

### 2.3 场景三：查看/管理调谐数据
用户想查看已存储的调谐数据：
1. 打开 ATU 调谐界面
2. 查看 "调谐数据" 标签页
3. 看到所有已存储的频率-SWR 映射
4. 可以删除无效数据或导出数据

### 3.3 调谐算法

#### 3.3.1 初始参数选择
```
1. 查找目标频率附近已有记录（±50kHz）
2. 如果有记录：
   a. 选择 SWR 最低的记录作为初始参数
   b. 设置继电器：sw, ind, cap
3. 如果无记录：
   a. 调用 ATR-1000 自动调谐 (mode=2)
   b. 等待调谐完成
   c. 获取当前继电器状态
```

#### 3.3.2 微调优化
```
对于每个频率点：
1. 设置初始参数
2. 发射低功率载波 (TUNE 模式)
3. 读取当前 SWR
4. 微调优化：
   a. 电感 L 微调：当前值 ±20% 范围内搜索
   b. 电容 C 微调：当前值 ±20% 范围内搜索
   c. 找到 SWR 最低的组合
5. 保存最佳参数到映射表
6. 移动到下一个频率点
```

#### 3.3.3 微调范围计算
```
ind 范围：0-127
  - 当前 ind = 50
  - 搜索范围：max(0, 40) 到 min(127, 60)
  - 步进：5 (避免过多测试)

cap 范围：0-127
  - 当前 cap = 30
  - 搜索范围：max(0, 24) 到 min(127, 36)
  - 步进：3
```

### 3.4 实时进度显示

#### 4.2.3 调谐状态机
```
状态：
- IDLE: 空闲
- TUNING: 正在调谐
- PAUSED: 暂停
- COMPLETED: 完成
- ERROR: 错误
```

### 4.3 安全考虑

#### 4.3.1 发射功率
- 使用 TUNE 模式低功率载波
- 建议功率：10-20W
- 避免长时间大功率发射

#### 4.3.2 调谐间隔
- 每个频点调谐后等待 500ms 让设备稳定
- 避免过快操作导致设备过载

#### 4.3.3 异常处理
- 设备断开连接时停止调谐
- SWR 过高 (>3.0) 时跳过该频点
- 支持随时中断调谐过程

### 7.2 ATR-1000 继电器参数说明
```
SW (网络类型):
  0 = LC 网络 (串联电感，并联电容)
  1 = CL 网络 (串联电容，并联电感)

IND (电感索引): 0-127
  - 实际电感值由设备内部表计算
  - 索引值需要 ×10 后发送

CAP (电容索引): 0-127
  - 实际电容值由设备内部表计算
  - 直接发送索引值
```
