## 二、核心理论：阻抗与 Counterpoise 的定量关系

### 2.1 MININEC 建模结果 (Fig.3/6)

AA5TB 用 MININEC 建立了完整模型：末端馈电半波导线 (~0.47λ free-space 谐振长度，此时 jX=0) + 可变长度 counterpoise。关键数据：

| Counterpoise 长度 | 馈电点阻抗 (Ω) | 电抗 (jX) | 状态 |
|-------------------|---------------|-----------|------|
| **~0.05λ** | **~1,800** | X≈0 | ✅ **最佳操作点**：纯电阻、稳定 |
| ~0.10λ | ~2,000–2,200 | 微感 | 过渡区 |
| ~0.25λ | ~2,500–3,500 | 过渡 | 传统 GP 思维误用至此 |
| ~0.38λ | ~4,000–4,500 | 高感 | 不稳定，环境敏感 |
| **~0.47λ** | **~5,000** | X≈0 | ⚠️ 纯电阻但极不稳定 |

### 2.2 "0.05λ Sweet Spot" 的物理本质

AA5TB 发现并验证了为什么 **0.05λ 是 counterpoise 最佳长度**：

```
物理模型（Moxon 等效分析，Fig.8）：

  [馈电点] ─── 0.05λ counterpoise ─── [自由空间高阻抗]
       │
       └──── ~0.47λ 辐射体 ───────── [自由空间高阻抗]
```

1. **半波辐射体两端都"看见"高阻抗** → 对 RF 电流而言，系统本质上是**平衡的**
2. **Counterpoise 的作用是提供低阻抗"参考点"**，而非传统意义上的"地网电流回路"
3. **0.05λ 刚好够提供低阻抗路径**，但又短到不会变成谐振辐射体（避免破坏平衡）
4. **更长的 counterpoise（包括 1/4λ）反而引入额外谐振**，可能增加馈线共模电流

### 2.3 对 "无 Counterpoise" 说的回应

AA5TB 明确回应了一个常见争论：

> **实践者说**："我不用 counterpoise，EFHW 照样工作！"
> **AA5TB 说**："对的——因为你总有寄生 counterpoise。"

任何实际安装中，以下至少一项充当了 de facto counterpoise：
- 同轴电缆外皮/屏蔽层
- 电台金属外壳
- 操作员人体电容
- 馈线接地系统
- 周围物体的杂散电容

**AA5TB 的洞察**：问题不是"是否需要 counterpoise"，而是"counterpoise 是否**可控**"。一个有意设计的 0.05λ 短 counterpoise 比依赖不确定的寄生路径可靠得多。

## 三、匝数比与阻抗匹配数学

### 3.1 基本变换公式

\[
\text{Turns Ratio} = \sqrt{\frac{Z_{\text{ant}}}{50}}
\]

| 匝数比 (N:1) | 阻抗比 (N²:1) | 匹配 Z_ant (Ω) | 推荐台架测试电阻 |
|-------------|--------------|----------------|-----------------|
| **6:1** | 36:1 | **1,800** | 1.8 kΩ (0.05λ counterpoise) |
| **7:1** | 49:1 | **2,450** | 2.2 kΩ 或 2.7 kΩ |
| **8:1** | 64:1 | **3,200** | 3.3 kΩ |
| 9:1 | 81:1 | 4,050 | 3.9 kΩ |
| 10:1 | 100:1 | 5,000 | 4.7 kΩ (0.47λ counterpoise) |

### 3.2 AA5TB 的推荐

- **主要推荐：6:1–8:1** — 对大多数实际安装最合适
- **Fig.6 SWR 稳定性分析**：**8:1 的 SWR 随 counterpoise 长度变化最小**—这是最重要的工程洞察之一
- **10:1 实测也工作**，但要求 counterpoise 非常接近 0.47λ（不切实际）
- **关键矛盾识别**：固定匝比只能匹配一个特定阻抗点 → 环境变化（高度、地面、临近物体）改变实际 Z_ant → SWR 随之变化。这就是为什么 **AA5TB 强烈推荐可调耦合器而非固定变压器**。

## 四、并联 LC 耦合器设计（核心工程方案）

### 4.1 电路拓扑

```
         L1 (初级链路, ~2–4T)
50Ω ────////////────  
  (馈线)  │    │        磁耦合
          │    │    ┌──────────────┐
          L2  === C │  天线 ~Z_ant │
         (~8–28T)===│  (1.8–5.0kΩ) │
          │    │    └──────────────┘
          │    │
     Counterpoise
      (~0.05λ)
```

**电路为经典并联谐振 tank circuit + 链路耦合（非变压器耦合）**：

- **L2 + C** = 并联谐振回路
  - 在谐振频率上，回路呈现**极高阻抗**（理论上 Q×X_L，实际 loaded Q ≈ 3–10）
  - 与天线的高阻抗 (~1,800–5,000Ω) 形成阻抗匹配
  - 提供天然的谐波滤波（>20 dB 二次谐波抑制）
- **L1** = 低阻抗链路（link coupling）
  - 通过互感将 50Ω 源的能量馈入高阻抗谐振回路
  - L1:L2 匝数比设置基础阻抗变换比
  - **与自耦变压器 (autotransformer) 不同**：物理隔离馈线/天线，显著减少共模电流
- **C** = 可变电容（调谐元件）
  - 调节谐振频率以匹配工作频段
  - 补偿环境变化导致的阻抗漂移

### 4.2 与 49:1 铁氧体变压器的根本区别

| 维度 | AA5TB 并联 LC 耦合器 | 现代 49:1 变压器 |
|------|---------------------|------------------|
| **匹配原理** | 并联谐振 + 链路耦合 | 磁通耦合自耦变压器 |
| **带宽** | 窄带（需调谐） | 宽带（免调谐） |
| **磁芯材料** | 空气芯或粉末铁芯 (Type 2/6) | Mn-Zn/Ni-Zn 铁氧体 |
| **高频损耗** | 极低（粉末铁芯涡流小） | 随频率升高增大 |
| **馈线-天线隔离** | 有（独立初级/次级） | 无（共地自耦） |
| **谐波抑制** | >20 dB（LC 滤波） | 无 |
| **操作便利性** | 需手动调谐 | 即插即用 |
| **多波段效率** | 每波段单独调谐 ≈ 恒定效率 | 低频电感不足、高频容性崩溃 |

### 4.3 组件要求

**电感/磁芯选择**：
- 空气芯（coil stock）：Q 最高，无磁芯损耗，适合固定频段 QRO
- 粉末铁芯 (Type 2, μ=10)：在 HF 全段（1.8–30 MHz）保持极低损耗，AA5TB 使用 **T-50-2** 和 **T-200-2**
- **不使用铁氧体**：Mn-Zn 铁氧体 (Type 43/31/61) 在 10–30 MHz 的磁滞损耗和涡流损耗显著增加

**可变电容要求**（⚠️ 关键安全考量）：

| 功率 | 类型 | 极板间距 | 额定电压 | 备注 |
|------|------|---------|---------|------|
| ≤10W QRP | Polyvaricon (塑料薄膜) | N/A | 150–300V | AA5TB mini coupler 使用 |
| ≤25W QRP | AM 收音机空气可变 (365pF) | 0.2–0.5mm | ~300V DC | 轻微超规格，需谨慎 |
| **≤100W** | **发射级空气可变** | **≥1.0mm** | **≥2,000V** | 从旧电子管功放拆机 |
| >100W | 真空可变电容 | N/A (真空介质) | 3–15kV | 最佳但昂贵，~¥50–200 |

**电容电压计算**：
- 并联谐振回路有 **Q 倍升压效应**：V_cap = Q × V_applied
- 100W @ 2,500Ω 次级 → V_applied ≈ 447V RMS
- 实际 loaded Q ≈ 3–10 → V_cap_peak = 1,900–6,320V
- **QRP 安全，100W 必须用发射级/真空电容** — 参见 [[efhw_ac_dual_mode]] §3.4 的定量分析

## 五、AA5TB 标志性调谐方法

这是 `efha_wrk.html`（"How to Make An End Fed Half Wave Antenna Work"）的核心内容，也是 AA5TB 页面在社区中被引用次数最多的方法论。

### 5.1 方法论流程

```
步骤 1: 台架标定 (Bench)
┌──────────────────────────────────────────┐
│  耦合器输出端接非感性电阻                │
│  (如 3.3kΩ @ 8:1 或 4.7kΩ @ 10:1)      │
│  → 用天线分析仪 (MFJ-259B) 调谐 C       │
│  → 找到 SWR=1:1, X=0 的位置             │
│  → ✏️ 标记电容刻度盘位置                 │
└──────────────────────────────────────────┘
           │
           ▼
步骤 2: 架设天线 (Deploy)
┌──────────────────────────────────────────┐
│  半波导线 (468/f MHz 英尺, 143/f MHz 米) │
│  + ≥0.05λ counterpoise (15/f MHz 米)    │
│  导线故意留长一点 (约 2–5%)             │
│  在远端做成可调环路/系结                 │
└──────────────────────────────────────────┘
           │
           ▼
步骤 3: 对比调谐 (Compare)
┌──────────────────────────────────────────┐
│  移除电阻, 接上天线                      │
│  → 重新用可变电容找谐振点               │
│  → 对比电容位置与步骤 1 标记            │
└──────────────────────────────────────────┘
           │
    ┌──────┼──────┐
    ▼      ▼      ▼
 需要更多C  需要更少C  电容位置不变
 天线太短    天线太长    ✅ 完美!
 放长导线    缩短导线   调谐完成
           │
           ▼
步骤 4: 迭代 (Iterate)
  调整导线长度 → 重新对比 → 直到电容位置与台架标定一致
```

### 5.2 物理学原理

当导线长度调整到使可调电容回到台架标定位置时：

1. **天线在馈电点呈现纯电阻**（X ≈ 0）— 导线精确谐振
2. **电阻值匹配耦合器设计阻抗**（如 3,200Ω @ 8:1）
3. **Counterpoise 电流最小化** — 因为导线是高阻抗电压节点，回流电流极小
4. **触摸电台/馈线不会改变 SWR** — 这是 **EFHW 正确调谐的标志**
5. **共模电流最小化** — 因为馈电点没有多余的 RF 电流"泄漏"到馈线外皮

### 5.3 现场经验法则

| 电容偏移方向 | 导线状态 | 修正方式 |
|-------------|---------|---------|
| 更多 C → 谐振 | **太短** | 放长（远端的 loop-back 结松开一点） |
| 更少 C → 谐振 | **太长** | 缩短（拉紧 loop-back 结） |
| 不变 | **完美** | 收工！ |

**关键洞察**：这个方法绕过了 EFHW 现场调谐最大的困难——你不需要知道天线在某个高度、某个环境下的精确阻抗是多少。你只需要让导线长度**追上台架已标定的耦合器**即可。

## 七、天线演化谱系 (Fig.9–14)

AA5TB 用约 5 张图展示了 EFHW 概念从经典天线中的演化，这是页面中最有价值的理论部分之一：

### 7.1 演化路径

```
阶段 1: 传统 1/2λ 中心馈电偶极子
   │  馈电点阻抗 ~50–73Ω (电流波腹)
   │
   └──→ 阶段 2: 末端馈电 1/2λ Zepp
        │  Zeppelin 天线 (齐柏林飞艇拖曳天线)
        │  匹配: 1/4λ 开路双线 → 端阻抗极高
        │  问题: 1/4λ 匹配段需要精确长度, 笨重
        │
        └──→ 阶段 3: J-Pole (1/4λ 短路 stub)
             │  匹配: 1/4λ 短路短截线, 在适当位置抽头
             │  优势: 紧凑, 坚固, 自支撑
             │  问题: 窄带, 单波段
             │
             └──→ 阶段 4: Slim-Jim (折叠 J-Pole)
                  │  DC 接地, 更宽带宽
                  │
                  └──→ 阶段 5: AA5TB LC 耦合器 EFHW
                       │  匹配: 集中参数 LC 谐振回路 (替代 1/4λ 分布参数线)
                       │  优势: 小型化, 可调谐, 多波段, 效率高
                       │
                       └──→ 阶段 6: 49:1/64:1 宽带变压器 EFHW
                            匹配: 铁氧体磁芯宽带自耦变压器
                            优势: 免调谐, 即插即用
                            代价: 高频效率下降, 无谐波滤波
```

### 7.2 关键洞察：集中参数 LC 取代分布参数线

AA5TB 设计的本质创新在于：

> **用一个可调谐的集中参数 LC 并联回路，等效替代了 Zepp 天线的 1/4λ 匹配段和 J-Pole 的 1/4λ 短截线。**

- 分布参数匹配线：物理长度固定 → 只能在一个频率 ± 少量偏移工作
- 集中参数 LC 回路：调 C 改变谐振频率 → **理论上可在任意 HF 频段工作**（受限于电感 Q 和电容范围）

这是 AA5TB 耦合器实现"多波段"的理论基础。

## 八、与现有项研究深度整合

### 8.1 AA5TB 耦合器在我们方案矩阵中的定位

来自 [[deep-dive-2026-05-25]] 和 [[efhw_ac_dual_mode]] 的分析：

```
              高效 ↑
                   │  AA5TB LC 耦合器   │  A+C Unit C       
                   │  (窄带,需调谐)     │  (T200-2 粉末铁芯)  
                   │  QRP: 无争议最优   │  高频段 >90%       
    ───────────────┼───────────────────┼──────────────────→ 宽带
                   │  DMEGC Ni-Zn      │  49:1 FT240-43     
                   │  (中宽带,中效)    │  (宽带,低频好)      
                   │  65–80% @ 80-10m  │  80m 差, 10m 崩溃  
                   │                    │                    
              低效 ↓
```

AA5TB LC 耦合器占据 **"窄带高效率"** 象限，在以下场景优于所有宽带方案：
- 固定频段数字模式（FT8、WSPR）
- 单波段竞赛
- 对电台谐波抑制有要求的场合

### 8.2 VA3KOT 实验验证 (2026-02) 与 AA5TB 原始设计的对照

来自 [[README#aa5tb-parallel-l-c-coupler]] ：

| 参数 | AA5TB 原始 | VA3KOT 复现 | 差异 |
|------|-----------|------------|------|
| 磁芯 | 多种 (空气芯, T-50-2) | **T-200-2** (更大截面积) | 升级 |
| 初级 | 2–4T | **3T** | 一致 |
| 次级 | 8–28T (可调) | **19T** (固定) | 简化 |
| 匝比 | 6:1–10:1 (可变) | **6.3:1** (固定) | 约 40:1 阻抗比 |
| 电容 | 多种 (固定/可变) | 16–160pF polyvaricon | 一致 |
| 台架测试电阻 | 按匝比选择 (1.8–4.7kΩ) | **2.7 kΩ** | 对应 7:1 → ~2,450Ω |
| 40m 结果 | 优秀 | SWR < 1.5:1, X=0 | ✅ 验证 |
| 30m 结果 | 优秀 | SWR < 1.5:1, X=0 | ✅ 验证 |
| 20m 结果 | 优秀 | SWR < 1.5:1, X=0 | ✅ 验证 |
| 17m 结果 | 未测试 | SWR < 1.5:1 | ✅ 验证 |
| 80m 结果 | 未明确 | **无法调谐** | ❌ 电感不足 |

### 8.3 A+C 双模系统的 AA5TB 基因

[[efhw_ac_dual_mode]] 中的 Unit C (高频匹配单元) 直接继承了 AA5TB 设计：

```
Unit C 设计参数：
- 磁芯: T200-2 (Type 2 粉末铁芯)
- 初级: 3T, 次级: 19T → 6.3:1 匝比
- 电容: 可变 (polyvaricon 或发射级空气可变)
- 目标频段: 30m–10m (Unit A 覆盖 80m–40m)

这是 AA5TB 设计在现代多波段 EFHW 中的直接应用。
```

我们在 A+C 分析中识别出的限制（[[efhw_ac_dual_mode]] §3.1–§3.8）**均来自 AA5TB 原始设计的固有物理约束**：

| 限制 | AA5TB 原始 | A+C 中的表现 | 严重程度 |
|------|-----------|-------------|---------|
| 可变电容耐压 | QRP 安全, QRO 需发射级 | Unit C polyvaricon @ 100W → 打火风险 | ❌ 严重 |
| 固定匝比 vs 可变的 Z_ant | 通过可变耦合 (tap) 补偿 | 3:19 固定 → 20m/10m 低阻抗不匹配 | ❌ 严重 |
| Counterpoise 多波段兼容 | 单波段设计，每波段独立 counterpoise | 4m 共用 → 10m 波段 0.38λ 变成辐射体 | ⚠️ 中等 |
| 户外可靠性 | 非设计目标（QRP 便携，室内存储） | IP65 壳内凝露 → 电容氧化/打火 | ❌ 严重 |
| 80m 能力 | 需更大电感（更多匝数或更大磁芯） | VA3KOT 未成功 | ⚠️ 可解决 |

## 九、AA5TB 理论的局限性与未解答的问题

### 9.1 已确认的局限

1. **单波段优化 vs 多波段折衷**
   - AA5TB 设计本质上是单波段优化的
   - 多波段使用需要重新调谐 C → 不适用于"切换频段即通联"的场景
   - 固定 20m 导线在不同频段呈现的电长度不同（参见 [[efhw_ac_dual_mode]] §3.8 表格）

2. **非半波频段的阻抗崩塌**
   - 20m 导线在 14.1 MHz = 0.94λ → **电流波腹，低阻抗 (~50–300Ω)**
   - 10m 导线在 28.5 MHz = 1.90λ → **电流波腹，低阻抗 (~50–300Ω)**
   - LC 耦合器的 40:1 阻抗比 (约 2,000Ω) 与 50–300Ω 严重不匹配 → **高 SWR，即使调到谐振**

3. **可变电容是系统最弱环节**
   - 机械可靠性有限（转子触点氧化、极板变形）
   - 电压额定值在实际使用中常被低估
   - 户外长期可靠性（凝露、灰尘、热循环）未经验证

4. **AA5TB 未涉及铁氧体材料的最新进展**
   - DMEGC Ni-Zn (μ=1000, Ae=350mm²) — 见 [[dmegc_nizn_toroid]]
   - 纳米晶磁芯 (nanocrystalline) — μ=500–2000, 高 Bs, 低损耗
   - 这些新材料可能在某些场景下提供宽带 + 中高效的替代方案

### 9.2 仍待验证的问题

| 问题 | 目前的了解 | 需要做什么 |
|------|-----------|-----------|
| T200-2 在 100W 100% 占空比 (FT8) 下的长期温升 | 理论估算 ~3–5W 损耗 | 热电偶实测 |
| Polyvaricon 在户外 IP65 壳中的寿命 | 理论预测 6–24 月 | 加速老化测试 |
| 粉末铁芯 LC 耦合器 vs FT240-43 49:1 的 A/B SNR 对比 | 无数据 | 同场 WSPR SNR 对比测试 |
| AA5TB 拓扑在 160m 的可行性 | 未尝试 | 大磁芯 + 大电感 + 大电容实验 |

## 十、结论与建议

### 10.1 AA5TB 方案的最佳应用场景

| 场景 | 推荐度 | 理由 |
|------|--------|------|
| QRP 便携单波段 (≤10W) | ⭐⭐⭐⭐⭐ | 最优方案 — 轻量、高效、polyvaricon 安全 |
| QRP 便携多波段 (手动调谐) | ⭐⭐⭐⭐ | 需每次调 C，但效率恒定高 |
| 固定频段数字模式 FT8/WSPR | ⭐⭐⭐⭐ | 可用固定电容 (高压) → 免维护 |
| 100W SSB/CW 固定频段 | ⭐⭐⭐ | 需发射级可变电容 → 成本/复杂性增加 |
| 100W 多波段快速切换 | ⭐⭐ | 每次手动调 C 不切实际 |
| 户外永久安装 | ⭐⭐ | 可靠性问题未解决 |

### 10.2 与 49:1 铁氧体变压器的选择决策

```
选择 AA5TB LC 耦合器 当：
  ✅ 你在固定频段操作
  ✅ 你需要最高 TX 效率
  ✅ 你需要谐波滤波 (>20 dB)
  ✅ 你愿意每次调谐
  ✅ 你是 QRP 操作者 (安全余量大)

选择 49:1 铁氧体变压器 当：
  ✅ 你需要快速切换频段
  ✅ 你的操作模式是搜索-呼叫 (S&P)
  ✅ 你需要即插即用
  ✅ 你使用的是 100W 级别
  ✅ 你接受 80m 和 10m 的性能折衷
```

### 10.3 对 BG1SB 的操作建议

基于 [[deep-dive-2026-05-25]] 和本分析：

- **FT8 主力场景**：AA5TB 单波段 LC 耦合器 + 固定高压电容 → **效率最优，零维护**
- **S&P / 多波段扫描**：DMEGC Ni-Zn 宽带变压器 → 免调谐 + 各波段可接受效率
- **终极方案**：TX/RX 分离 ([[deep-dive-2026-05-25]] §2.3) — EFHW TX + 有源磁环 RX
- **A+C 双模**：等待工程成熟（高压切换、电容可靠性解决后值得尝试）
