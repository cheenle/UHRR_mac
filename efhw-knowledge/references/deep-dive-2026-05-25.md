## 二、关键发现：EFHW 的真正瓶颈是 RX，不是 TX

### 2.1 社区共识转向

经过 2025-2026 年的深入讨论，Ham 社区正在形成新的共识：

> **EFHW 的发射性能"足够好"——真正的短板是接收端对 QRM 的极度敏感。**

- N6CC (2026-03): "被听到但听不到回复"是 EFHW 的首要痛点
- PA9X (2025-08): 整篇文章专门讨论 EFHW + QRM 解决方案
- VA3KOT (2026-03): 承认 EFHW 在便携场景下无可替代

### 2.2 物理原因

EFHW 天线呈现极高的馈电点阻抗（~2500Ω），高阻抗节点天然对电场噪声敏感：
- 末端馈电 = 电压波腹 = 电场最大点
- 对近场电器噪声极度敏感（LED灯、开关电源、PLC）
- 同轴电缆外皮上的共模电流将整个房子的地线系统变成天线

### 2.3 解决方案：TX/RX 分离架构

PA9X 推荐的黄金组合：

```
┌─────────────────────────────────────────┐
│  TX 路径：EFHW + 49:1 变压器              │
│  → 多波段发射，简单高效                    │
│  → 用现有天线，不做改动                    │
├─────────────────────────────────────────┤
│  RX 路径：有源磁环天线 (Active Loop)       │
│  → 噪声基底低 10-20dB                    │
│  → 可旋转null对准噪声源                   │
│  → 直径 < 1m，小巧隐蔽                    │
├─────────────────────────────────────────┤
│  T/R 开关：自动切换                        │
│  → QRO.cz RX Antenna Switch             │
│  → W6LVP Loop + T/R (一体方案)           │
│  → 电台内置RX ANT接口（IC-7300等）         │
└─────────────────────────────────────────┘
```

**推荐磁环天线方案：**

| 方案 | 价格 | 性能 | 适合 |
|------|------|------|------|
| **MLA-30+** (国产) | ~¥150 | 入门级 | 预算有限、试验 |
| **Wellgood Loop v4** (M1GEO) | ~¥300 (PCB+零件) | 中等 | DIY能力强的火腿 |
| **W6LVP 完整方案** | ~$300 | 专业级 | 一步到位 |
| **Cross Country Wireless** | ~£100 | 中高端 | 欧洲火腿 |

**关键技巧：将磁环装在简易旋转器上，把 null 对准噪声源方向。**

## 三、变压器设计深度对比

### 3.1 三种方案对比

| 方案 | 效率 | 带宽 | 复杂度 | 便携 | 成本 |
|------|------|------|--------|------|------|
| **49:1 铁氧体变压器** | 60-85% | 多波段 | 低 | ⭐⭐⭐ | ~¥50 |

### 3.2 铁氧体 49:1 变压器：分层设计建议

基于知识库+新研究，按功率等级推荐：

```
║         BG1SB EFHW 工程决策树                ║
╠══════════════════════════════════════════════╣
║                                              ║
║  QRP (<20W) 便携?                            ║
║    ├─ YES → FT140-43, 2T:14T 并绕           ║
║    │        轻量、简单、足够                  ║
║    │                                         ║
║  QRO (<100W) 固定台站?                       ║
║    ├─ 80m 主力?                              ║
║    │   ├─ YES → DMEGC Ni-Zn μ=1000          ║
║    │   │    50×25×28mm, 2T:14T 并绕         ║
║    │   │    80m损耗=FT240-43的43%            ║
║    │   │                                     ║
║    │   └─ NO → FT240-43, 2T:14T 并绕        ║
║    │          40m-10m 性价比最高              ║
║    │                                         ║
║  追求极致效率 (单波段便携)?                   ║
║    └─ L-Network: 可调电感 + 陶瓷电容          ║
║       无磁芯损耗，效率 >90%                   ║
║                                              ║
╚══════════════════════════════════════════════╝
```

### 3.3 统一绕线规范（全方案推荐）

> **W8JI 实测结论：HF 频段并绕法优于分绕法。以下为推荐标准：**

```
磁芯：FT240-43 / DMEGC Ni-Zn / FT140-43
绕法：简单并绕 (W8JI 验证)
线圈：2T 初级 (0.8-1.0mm 漆包线，外层散热)
     + 14T 次级 (0.5-0.6mm 漆包线)
阻抗比：(14/2)² = 49:1 ✓

具体绕制步骤：
1. 初级 2T 均匀绕在磁芯外层
2. 次级 14T 紧贴初级并绕，匝间紧密
3. 初次级线并行于同一区域（不分离）
4. 次级冷端接 counterpoise 端子（不可省略！）
5. 整体做防水处理（户外使用）
```

## 五、FT8/Digital Mode 特别注意事项

### 5.1 占空比与散热

FT8 发射周期：15秒 TX / 15秒 RX = 50% 占空比
连续呼叫时接近 **100% 占空比**（每周期都发射）

```
FT8 连续发射 30分钟 → 相当于 FM 模式 15分钟持续功率
                  → 磁芯温升更高！
```

**建议：**
- FT240-43：100W FT8 安全，但 80m 波段注意温升（N6CC实测石蜡未融，但仍需警惕）
- DMEGC Ni-Zn μ=1000：截面积 2.2x，温升更安全（计算 <22°C @100W FT8 连续）
- FT140-43：FT8 功率限制在 30W 以下
- **FT8 高占空比 = 不要只看 SSB PEP 功率，用平均功率规划散热**

### 5.2 功率合成器场景

BG1SB 使用功率合成器时：
- 多个发射机的功率合成 → 变压器承受叠加功率
- 5W+5W 合成 → 变压器等效承受 ~10W（但仍要按 20W 保守设计）
- **建议使用更大磁芯**（FT240而非FT140），即使单路功率不高

## 六、Benchmark：各方案预期性能

### 6.1 以 40m 基波为基准

| 配置 | 辐射效率 | 多波段 | 架设时间 | QRM底噪 | 适用场景 |
|------|---------|--------|---------|---------|---------|
| EFHW 49:1 (FT240) 单独 | 60-75% | 40/20/15/10 | 5min | S7-S9 | 野外快速架设 |
| EFHW 49:1 (DMEGC) 单独 | 65-80% | 40-10m + 80m | 5min | S7-S9 | 80m主力便携 |
| EFHW + 磁环 RX | 60-75% TX / 极高 RX | 40/20/15/10 | 10min | **S3-S5** 🏆 | 城市/郊区固定台 |
| L-Network 单波段 | 85-95% | 单波段 | 15min | S7-S9 | 极致效率便携 |
| 偶极子 (CFHW) | 85-90% | 单波段 | 20min | S5-S7 | 固定台/宽松场地 |

### 6.2 BG1SB 最佳配置推荐

> **场景：北京城市/郊区固定台站，FT8 为主，已有 IC-M710 + 功率合成器，偶有便携需求**

**推荐：分体式方案**

```
固定台站：
├── TX: 40m EFHW + DMEGC Ni-Zn 49:1 变压器 (2T:14T 并绕)
│       Counterpoise 2m, CMC FT240-43 7圈 RG-58 @3m
├── RX: MLA-30+ 有源磁环 (或 Wellgood v4 自建)
│       装旋转器，null对准主要噪声源
└── T/R: QRO.cz RX Switch 或电台内置 RX ANT

便携操作：
└── EFHW + FT140-43 QRP 变压器 (轻便版)
    同上 counterpoise/CMC 配置，缩小版
```
