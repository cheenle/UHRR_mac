## 一、方案概述

### 1.1 核心思路

传统 49:1 EFHW 变压器在 80-10m 全段面临两个不可调和的矛盾：
- **低频端（80m）**：初级电感不足，磁化电流过大 → 磁芯发热，效率下降
- **高频端（10m）**：匝间寄生电容主导，变压器变成容性 → RF 传输恶化，效率崩溃

A+C 双模方案的解决思路：**低频用铁氧体（宽带免调谐），高频用粉末铁芯（极致效率）。**
两套匹配网络集成在同一馈电盒中，通过内置 DPDT 开关在天线侧切换。

## 二、理论验证：为什么这个方向有道理

### 2.1 铁氧体 vs 粉末铁芯的物理本质

| 特性 | 铁氧体 (Mn-Zn Type 43) | 粉末铁芯 (Type 2) |
|------|------------------------|-------------------|
| μ (磁导率) | 850 | 10 |
| 电阻率 | 1-10 Ω·m | 每个颗粒绝缘 → 极高 |
| 高频涡流损耗 | 随频率显著增大 | **极低** |
| 温度稳定性 | 一般（μ 温漂大） | **优秀** |
| 饱和磁通密度 | 低（~0.3T） | **高（~0.8-1.0T）** |
| 分布式气隙 | 无 | 有（颗粒间绝缘层） |
| 适用场景 | 宽带变压器 | **窄带调谐电路** |

**关键物理机制**：粉末铁芯的每个微观铁粉颗粒都被绝缘层包裹，形成"分布式气隙"。这带来三个优势：
1. 涡流被限制在单个颗粒内 → 高频损耗极低
2. 分布式气隙使磁芯极难饱和 → 高功率容量
3. μ 低但稳定 → 频率响应平坦，不会像铁氧体那样 μ 随频率暴跌

**代价**：μ 只有铁氧体的 1/85，需要更多匝数达到相同电感 → 窄带调谐方案的必要性。

### 2.2 为什么单一变压器方案在 80-10m 注定失败

RF.Guru (ON6URE) 的 VNA 实测和 N6CC、PA3HHO 等独立来源一致确认：

```
频率        单一 FT240-43 49:1 变压器的问题
3.5 MHz     初级感抗仅 ~94Ω (2T)，远低于 DL1CR 经验法则的 ≥200Ω
            磁化电流过大 → 磁芯发热 → 能量浪费
7 MHz       最佳工作点
14 MHz      尚可，损耗开始上升
18-24 MHz   匝间寄生电容开始主导，变压器不再"变换"阻抗
28 MHz      容性主导，效率崩溃至 ~55%
```

A+C 方案正确地识别了这个问题，并提出分频段用不同磁芯材料的解决思路——这在理论上完全成立。

### 2.3 AA5TB 并联谐振 LC 耦合器的工程原理

> 📄 **完整参考**: [[aa5tb_efha_analysis]] — AA5TB 原始理论体系全解 (MININEC, 0.05λ 法则, Moxon 等效, 台架调谐法, 演化谱系)

AA5TB (Steve Yates) 的原始设计利用了并联谐振电路的阻抗变换特性：

```
         L1 (3T 初级)
50Ω ────////////────  
         │    │       磁耦合
         │    │    ┌──────────┐
         L2  === C │ 天线     │
        (19T) ===   │ ~2500Ω   │
         │    │    └──────────┘
         │    │
     Counterpoise
```

- L2 与 C 构成并联谐振回路
- 在谐振频率上，回路呈现高阻抗，与天线的高阻抗匹配
- 能量通过磁耦合从 L1 传递到 L2
- **关键优势**：不依赖铁氧体的宽带变换，而是通过谐振实现窄带高效率匹配

VA3KOT (John) 在 2026-02 用 T200-2 磁芯 + polyvaricon 可变电容验证了该方案：
- 40m/30m/20m: 可调谐到 X=0（纯电阻），SWR < 1.5:1
- 17m/15m/12m/10m: 可调谐到 SWR < 1.5:1
- 80m: 无法调谐（电感不足）

## 三、关键工程问题与定量分析

### 3.1 DPDT 开关电压额定值 ⚠️ 中等风险

**方案声称**：开关承受 ~2500Ω @ 0.2A RMS，陶瓷旋转开关可胜任。

**定量分析**：
- 100W @ 2500Ω → V_peak = √(100 × 2500) × √2 = **707V**
- 典型陶瓷旋转开关额定 250VAC (354V peak)
- 空气间隙 ~2mm → 理论击穿电压 ~5000V peak
- **实际风险**：开关 OFF 状态的极间寄生电容 ~2pF @ 28MHz → Xc ≈ 2800Ω
  - 这个值与天线阻抗(~2500Ω)可比拟，形成并联泄漏路径

**结论**：干净干燥条件下可能工作，但额定值被超出 2 倍。建议改为 **50Ω 端切换**（V_peak 仅 100V），完全消除高压风险。

### 3.2 Counterpoise 长度与多波段兼容性 ❌ 严重问题

**方案声称**：4m counterpoise（80m 的 0.05λ）在所有波段共用。

**定量分析**：

| 波段 | 频率 | 波长 | 4m 占比 | 最佳 0.05λ | 实际状态 |
|------|------|------|---------|-----------|---------|
| 80m | 3.5 MHz | 85.7m | 0.047λ | 4.3m | ✅ 接近最优 |
| 40m | 7.0 MHz | 42.9m | 0.093λ | 2.1m | ⚠️ 偏长 |
| 20m | 14.1 MHz | 21.3m | 0.188λ | 1.1m | ⚠️ 显著偏长 |
| 10m | 28.5 MHz | 10.5m | **0.381λ** | 0.53m | ❌ 已变成辐射体 |

**物理机制**：Counterpoise 接近 0.25λ 时变成谐振 radial，开始辐射而非提供低阻抗回路。10m 波段 4m = 0.38λ，已远超"地网"范畴。

**后果**：
- 10m 波段 SWR 不可预测
- 馈线共模电流增加（CMC 下游可能被击穿）
- 方向图畸变（不对称辐射）

**改进建议**：Unit C 切换时同步切换到短 counterpoise（~0.5m），或使用 DPDT 第三刀（3P2T 开关）切换 counterpoise。

### 3.3 T200-2 磁芯饱和 ❌ 不是问题

**定量分析**：
- T200-2: Ae = 1.33 cm², μ = 10, AL = 12 nH/N²
- 30m 波段 (10.1 MHz, 最低 Unit C 频率)：
  - B_peak = 447V / (4.44 × 10.1e6 × 19T × 1.33e-4 m²) = **39.4 gauss**
- Type 2 材料饱和磁通 ~800-1000 gauss
- **39.4 gauss ≈ 4-5% 饱和值 → 巨大余量**
- 磁芯损耗估算：1.7W @ 10.1MHz, 3-5W @ 28MHz（热管理可行）

**结论**：T200-2 在 100W 下不存在饱和风险。磁芯发热 <5W，在 IP65 铝壳中可接受。

### 3.4 可变电容耐压 ❌ 严重安全风险

**方案声称**：空气可变电容 ≥1000V 额定值。

**定量分析**：
- LC 并联谐振电路中，电容器承受的电压 = Q × V_applied（谐振升压效应）
- V_applied ≈ 447V RMS (次级端)
- 实际 Loaded Q ≈ 3-10
- V_cap_peak 估算：
  - Q=3: ~1897V peak
  - Q=5: ~3160V peak
  - Q=10: ~6320V peak

**典型 AM 收音机拆机电容**：
- 最大容量 365pF，极板间距 ~0.2-0.4mm
- 额定电压 150-300V DC
- 0.3mm 间隙 → 空气击穿 ~900V peak
- 1897V / 0.3mm = **6.3 kV/mm > 3 kV/mm (空气介电强度) → 必然打火！**

**安全建议**：
- QRP (≤10W): polyvaricon 或普通空气可变电容可接受
- 100W: **必须使用真空可变电容**（3-15kV 额定，~¥50-200）或发射级空气可变电容（极板间距 ≥1mm, 2-5kV 额定，从旧电子管功放拆机）

### 3.5 两个 Unit 之间的隔离 ⚠️ 中等风险

**方案自认**："空闲 Unit C 的 LC 谐振回路在谐振频率仍会吸收能量"

**定量分析**：
- DPDT OFF 态极间电容 ~2pF → Xc @ 7MHz ≈ 11,400Ω
- Unit C 谐振时 Z_tank ≈ 19,000Ω (Q=100)
- 功率分配：~37.5% 电压降在 Unit C → 约 **1.86W 寄生吸收**
- 在 100W 系统中损耗约 2%，不致命但浪费

**电感耦合**：两个环形磁芯在同一铝壳中，耦合系数 k < 0.01（环形自屏蔽），电感耦合路径可忽略。

**改进建议**：空闲 Unit 的初级应**短路或端接 50Ω**（而非悬空），以阻尼谐振回路。需 3P2T 开关或继电器实现。

### 3.6 户外长期可靠性 ❌ 高风险

**IP65 不等于防水汽**：
- 昼夜热循环 → 壳内呼吸效应 → 湿气进入
- 露点冷凝 → 电容极板结露 → 电容值剧变 + 低压打火
- 铝壳 + 黄铜/铝电容极板 → 电偶腐蚀
- 转子滑动触点（磷青铜）→ 氧化 → 非线性结 → 谐波 + 噪声
- 预计户外寿命：**6-24 个月**（vs 固定变压器 5-10 年）

**缓解措施**：干燥剂包（定期更换）、真空可变电容、固定电容+波段开关方案。

### 3.7 与更简单方案的对比 ❌ 设计过度

| 方案 | 80m | 40m | 20m | 10m | 复杂度 | 成本 | 可维护性 |
|------|-----|-----|-----|-----|--------|------|---------|
| 单 FT240-43 2T:14T | 差 | 好 | 好 | 一般 | 极低 | ~¥50 | 优秀 |
| 2×FT240-43 3T:21T | 好 | 好 | 一般 | 一般 | 低 | ~¥100 | 优秀 |
| **DMEGC Ni-Zn μ=1000** | **好** | **好** | **好** | **好** | **低** | **~¥80** | **优秀** |
| A+C 双模 | 好 | 好 | 好 | 优秀 | **极高** | ~¥400-600 | **差** |

**DMEGC Ni-Zn (50×25×28mm, μ=1000, Ae=350mm²)** 是当前知识库中已分析的单一磁芯方案：
- 80m 初级 XL = 220Ω（达标），截面积 2.2× FT240-43
- Ni-Zn 电阻率是 Mn-Zn 的 10⁵-10⁶ 倍 → 10m 涡流损耗极低
- 无需开关、无需可变电容、无需波段切换、无需调谐
- 理论效率 65-80%（80-10m），虽不及 Unit C 的 90-95%，但可靠性和简易性远超

**对于 BG1SB 的 FT8 使用场景**：
- TX 效率差异 ~0.2-0.4 dB（不可感知）
- FT8 的真正瓶颈是 **RX 噪声基底**，不是 TX 效率
- 社区共识：TX/RX 分离（EFHW TX + 有源磁环 RX）是性价比最高的升级

### 3.8 20m 导线在高波段的阻抗特性 ❌ 根本性设计缺陷

这是 A+C 方案**最严重的理论问题**：

| 波段 | 频率 | 线长(λ) | 馈电点阻抗类型 | 预期间 |
|------|------|---------|---------------|--------|
| 80m | 3.5 MHz | 0.23λ | 中阻抗 (~200-500Ω) | Unit A |
| 40m | 7.0 MHz | 0.47λ | 高阻抗 (~2000-5000Ω) ✅ | Unit A |
| 30m | 10.1 MHz | 0.67λ | 中间态 (~800-2000Ω) | Unit C |
| **20m** | **14.1 MHz** | **0.94λ** | **低阻抗 (~50-300Ω)** ❌ | Unit C |
| 17m | 18.1 MHz | 1.21λ | 复杂（高电抗） | Unit C |
| 15m | 21.1 MHz | 1.41λ | 高阻抗 (~2000-4000Ω) | Unit C |
| 12m | 24.9 MHz | 1.66λ | 中间态 (~500-1500Ω) | Unit C |
| **10m** | **28.5 MHz** | **1.90λ** | **低阻抗 (~50-300Ω)** ❌ | Unit C |

**端馈线天线阻抗规律**：
- n × λ/2 → 电压波腹 → 极高阻抗 (2000-5000Ω)
- n × λ → 电流波腹 → 低阻抗 (50-500Ω)
- 非谐振长度 → 电抗主导

**关键矛盾**：AA5TB LC 耦合器（匝比固定 3:19 ≈ 40:1 阻抗比）只能高效匹配一个有限的阻抗范围（约 800-4000Ω）。20m 和 10m 波段导线接近整数倍波长，呈现**低阻抗**而非高阻抗。可变电容可以调谐频率（抵消电抗），但**无法补偿电阻分量 40:1 的巨大变化**。

这解释了为什么 VA3KOT 在 80m 失败——同样的物理限制。

**后果**：
- 20m 和 10m 可能出现高 SWR（即使能调到谐振，阻抗也不匹配）
- 或者以极低效率"匹配"（大 circulating current → 发热）

## 四、改进方案：A+C v2.0 工程建议

基于上述分析，提出以下改进方案来解决已识别的 8 个问题：

### 4.1 DPDT 改 50Ω 端切换（解决问题 1）

```
        天线 ─────┬────── Unit A 天线端
                  │
                  ├────── Unit C 天线端
                  │
           (天线端直接并联，不切换)
                  │
            ┌─────┴─────┐
            │   DPDT    │ ← 在 50Ω 端切换！
            │  50Ω 端   │   V_peak 仅 100V，安全
            ├─A──────C──┤
            │           │
        Unit A 50Ω   Unit C 50Ω
            │           │
            └─────┬─────┘
                  │
              SO-239
```

**优势**：电压从 707V 降至 100V，任何开关都能胜任。
**代价**：天线端始终并联两个 Unit，但 50Ω 端隔离更干净。

### 4.2 切换 Counterpoise 长度（解决问题 2）

使用 3P2T 开关（或 DPDT + 继电器），Unit A 用 4m，Unit C 用 0.5m counterpoise。

### 4.3 真空可变电容（解决问题 4, 6）

- 真空可变电容 5-50pF：额定 3-15kV，全密封 → 同时解决耐压和防潮
- 成本 ~¥80-200（二手/国产），但彻底消除打火和腐蚀风险
- 配合固定电容扩展范围

### 4.4 空闲 Unit 初级短路（解决问题 5）

3P2T 开关第三组触点：选中 Unit A 时短路 Unit C 初级 → 阻尼谐振 → 消除寄生吸收。

### 4.5 可切换匝数比（解决问题 8）

Unit C 的 3T 初级改为多抽头（2T/3T/4T），通过旋转开关选择：
- 20m/10m（低阻抗）：选 2T 初级（阻抗比 90:1）
- 15m（高阻抗）：选 3T 初级（阻抗比 40:1）

## 五、方案定位与实用建议

### 5.1 A+C 方案的合理定位

A+C 不是一个"开箱即用"的成熟方案，而是一个**有理论基础的工程探索**。它的核心洞察（分频段用不同磁芯材料）是正确的，但工程实现需要更多迭代。

### 5.2 分阶段实施路线

```
Phase 1 (立即, ¥80, 0风险):
  └─ DMEGC Ni-Zn μ=1000 单磁芯 49:1 变压器
     替换现有 FT240-43 → 全频段效率提升，无新增复杂度

Phase 2 (短期, ¥200, 低风险):
  └─ TX/RX 分离：MLA-30+ 有源磁环 + RX ANT 接口
     解决 EFHW 最大痛点（接收噪声基底 S7→S3）

Phase 3 (中期, ¥400, 中风险):
  └─ A+C v2.0 原型：DMEGC (Unit A) + T200-2 + 真空可变电容 (Unit C)
     50Ω 端切换，真空电容防潮，可切换匝比
     先在桌面用 NanoVNA + 假负载验证全波段可匹配性

Phase 4 (长期):
  └─ 户外部署 + 季度检查 + 干燥剂更换 + 冬季/雨季可靠性验证
```

### 5.3 一句话总结

> **A+C 方案代表了一个重要的探索方向：承认单一铁氧体变压器在 80-10m 的物理极限，用分频段多磁芯方案突破瓶颈。当前原型的工程实现有多个需要解决的实际问题（高压切换、可变电容可靠性、非谐振波段阻抗匹配），但这些问题是可解决的——需要的是工程迭代，而非放弃方向。**
