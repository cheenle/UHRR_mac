# 室内 Bias-T 直流注入盒 — 独立子设计
# =========================================
# 用途: 配合室外 EFHW Fuchs ATU V3.0 (ESP32-S3) 使用
#       通过同轴电缆同时传输 RF (100W) 和 DC (13.8V)
# 版本: V1.1 (2026-06-09)

---

## 一、设计目标

| 参数 | 规格 |
|------|------|
| RF 功率 | 100W PEP (1.8-30 MHz) |
| DC 电流 | 最大 3A (伺服堵转瞬态) / 空闲 ~80mA / 调谐 ~500mA |
| 插入损耗 | < 0.2 dB @ 1.8-30 MHz |
| DC-RF 隔离 | > 60 dB |
| 尺寸 | 50×50×30mm 小铝盒 |
| RF 接头 | SO-239 ×2 (输入/输出) |
| DC 输入 | 5.5/2.1mm DC插座, 中心正极 |

---

## 二、电路原理图

```
         Bias-T 注入盒

  [电台 TX OUT]                    [去室外同轴电缆]
   SO-239 (J1)                       SO-239 (J2)
       │                                │
       ├── C_block ────────────────────┤
       │    10nF / 1KV C0G             │
       │    1206 ×2 并联              │
       │                                │
       ├── L_rfc ──────────────────────┤ (连接DC)
       │    宽带RF扼流圈              │
       │    (见选型指南)              │
       │         │                     │
       │    C_bypass                   │
       │    100nF / 50V               │
       │    (RF 残余旁路到地)        │
       │         │                     │
       │    ┌────┴────┐               │
       │    │ DC 插座  │               │
       │    │ 5.5/2.1  │               │
       │    │ 中心正极 │               │
       │    └─────────┘               │
       │                                │
       └──── GND (铝壳) ───────────────┘
```

### 工作原理解释：

1. **RF 路径**：电台 TX → C_block (隔直, 通RF) → 同轴电缆出门
   - C_block 阻止 DC 回灌到电台输出级
2. **DC 路径**：13.8V 电源 → L_rfc (隔交, 通DC) → 同轴电缆芯线
   - L_rfc 阻止 RF 进入电源, 同时让 DC 通过
3. **C_bypass**：L_rfc 末端的对地旁路电容 → 进一步衰减可能耦合到 DC 线的残余 RF

## 三、关键器件选型

### 3.1 隔直电容 C_block

| 参数 | 选型 | 为什么 |
|------|------|--------|
| 容值 | 10nF ×2 并联 | XL@1.8MHz = 8.8Ω单只 / 4.4Ω并联 → 插损可忽略 |
| 耐压 | 1KV | 100W@50Ω → Vpeak≈100V, 1KV有10×裕度 |
| 介质 | C0G/NPO | 零老化, 零压电效应, 极低损耗 |
| 封装 | 1206 | 耐压 1KV 且 RF ESR 低 |
| 数量 | 2 只并联 | 降低 ESR/ESL, 分流 RF 电流 |

### 3.2 RF 扼流圈 L_rfc

这是 Bias-T 中**最关键**的元件。错误的电感选择会导致：
- SRF 落在工作频段内 → 电感变容性 → DC-RF 隔离崩溃
- 饱和电流不足 → 磁芯饱和 → 电感量骤降

**方案对比**：

| 方案 | 型号 | L (μH) | SRF (MHz) | I_DC (A) | 价格 | 推荐度 |
|------|------|--------|-----------|---------|------|--------|
| **DIY绕制** | FT37-43 上绕15T | ~95 | >100 | 2 | ¥5 | ⭐⭐⭐⭐⭐ |
| DIY绕制 | FT50-43 上绕12T | ~60 | >150 | 3 | ¥8 | ⭐⭐⭐⭐⭐ |
| 成品电感 | Coilcraft 4310LC-225 | 220 | 25 | 5.6 | ¥40 | ⭐⭐⭐⭐ |
| 成品电感 | Mini-Circuits ADCH-80A+ | — | 0.05-6000 | 0.5 | ¥60 | ⭐⭐⭐⭐ (最宽带) |
| 普通工字电感 | CD75 22μH/3A | 22 | ~15 | 3 | ¥2 | ⭐⭐ (SRF 仅15MHz) |

**强推方案：FT37-43 DIY 绕制**

```
材料: FT37-43 铁氧体磁环 ×1 (OD=9.5mm, ID=4.8mm, AL≈420nH/N²)
漆包线: 0.5mm 聚氨酯漆包铜线, 约 60cm

绕制:
1. 在 FT37-43 上均匀绕 15 圈
2. 电感量: AL × N² = 420nH × 15² ≈ 94.5 μH
3. XL @ 1.8 MHz = 2π × 1.8MHz × 95μH ≈ 1,074 Ω >> 50 Ω ✓
4. SRF: FT37-43 15T 的自谐频率 > 100 MHz (远高于 30MHz HF 段)
5. 可承受 DC 电流: FT37-43 在 ~100mA DC 偏流下电感量几乎不变
   (铁氧体有气隙分布, DC 磁偏能力远好于闭环磁芯)
```

用 nanoVNA 验证：
- `S21` (DC端口到RF输出端口) @1.8-30MHz → < -60dB (RF基本不通)
- `S11` (RF输入端, DC端口悬空) → > -0.1dB (RF顺利通过)

### 3.3 旁路电容 C_bypass

| 参数 | 选型 |
|------|------|
| 容值 | 100nF / 50V X7R |
| 封装 | 0805 |
| 作用 | 将可能通过 L_rfc 分布电容耦合过来的微量 RF 旁路到地 |

## 五、装配步骤

```
1. 铝盒两端钻 M座安装孔 (Ø16mm)
2. 铝盒侧壁钻 DC插座安装孔 (Ø12mm)
3. 安装两个 SO-239 M座 → 法兰从外拧紧
4. 安装 DC 插座 → 螺母拧紧
5. 绕制 L_rfc: FT37-43 上均匀绕 15T 0.5mm 漆包线
6. 内部接线 (尽量短):
   a. J1 芯线 → 两只 C_block 并联 (焊在 J1 芯线焊片) → J2 芯线
   b. L_rfc 一端接 J1 芯线焊片, 另一端接 DC 插座正极
   c. C_bypass 焊在 DC 插座正极与铝壳地之间
   d. L_rfc 用尼龙扎带固定在铝壳侧壁 (不接触金属短路!)
7. 所有地 (J1外壳, J2外壳, DC插座负极) → 铝壳本身
8. 合盖前用万用表验证:
   a. J1 芯线 ↔ J2 芯线: 导通 (通过 L_rfc 的DC路径)
   b. J1 芯线 ↔ DC 插座正极: ≈ L_rfc 直流电阻 (< 0.5Ω)
   c. J1 芯线 ↔ 铝壳: ∞ Ω (无短路)
   d. DC 插座正极 ↔ 铝壳: 100nF 电容 (由 C_bypass 接入)
```

## 六、测试步骤

### 6.1 DC 通路测试

```
1. DC 电源接 DC 插座 (13.8V, 电流限制 0.5A)
2. 万用表测 J2 芯线对壳: 应为 13.8V
3. 电流消耗 (无负载): < 1mA
```

### 6.2 RF 插入损耗测试 (nanoVNA)

```
1. VNA 校准 (1-30 MHz, S21 直通校准)
2. Port1 → J1 (电台侧), Port2 → J2 (出门侧)
3. DC 插座不接电源
4. 测量 S21: 应在 1-30 MHz 全段 < -0.2 dB
   如果 S21 > -0.5 dB → L_rfc 自谐频率落在 HF 段! 重新绕制。
```

### 6.3 DC-RF 隔离测试

```
1. VNA Port1 → J1 (电台侧)
2. J2 接 50Ω 终端
3. VNA Port2 → DC 插座正极 (通过隔直块!)
4. 测量 S21: 应在 1-30 MHz 全段 < -60 dB
   如果 > -40 dB → 扼流圈电感不足或饱和
```

### 6.4 满载测试

```
1. J2 接 50Ω/100W 假负载
2. DC 插座接 13.8V/0.3A 电源
3. 电台 → J1, CW 模式 100W, 14.100 MHz
4. 发射 30 秒 → 测量:
   a. DC 电源输出: 13.8V 稳定 (无压降、无波动)
   b. C_block/L_rfc 温升: 手摸不烫 (< 10°C)
   c. 电台 SWR: < 1.1:1 (Bias-T 本身不引入失配)
```

## 七、注意事项

1. **L_rfc 不能短路到铝壳** — 扼流圈一端接 M座芯线(有13.8V DC!), 另一端接 DC插座, 磁环本身必须悬空或非金属固定
2. **C_block 的耐压方向性** — C0G 电容无极性的, 但两只并联时确保焊接正确
3. **不要装反极性** — 如果 DC 电源反接, 同轴电缆芯线上出现负电压 → 室外 ATU 的 LM2940CT-12 会被 1N4007 防反接二极管保护, 但最好不要发生
4. **先用 5W 测试** — 第一次使用时, 用低功率验证一切正常, 再逐步升高到 100W
