## 2. 谐振频率覆盖分析

### 2.1 谐振方程

```
f_res = 1 / (2π × √(L2_eff × C_total))

L2_eff = L2 + L_leak    (次级有效电感, ~2.06 μH)
C_total = C_var + C_stray + C_winding
```

### 2.2 杂散电容估算

```
C_stray (PCB 走线 + 接线柱):  ~4-6 pF
C_winding (匝间电容):        ~3-5 pF (14匝 密绕, 0.5mm 线)
C_stray_total:               ~10 pF (保守)

C_total_min = 10 pF (可变电容最小值) + 10 pF (杂散) = 20 pF
C_total_max = 500 pF (可变电容最大值) + 10 pF (杂散) = 510 pF
```

### 2.3 频率覆盖表

| C_total (pF) | f_res (MHz) | 对应频段 | 备注 |
|:------------:|:-----------:|:--------:|------|
| 20 | **35.07** | 10m (28.0-29.7) | ✓ 覆盖, 有 5+ MHz 余量 |
| 30 | 28.66 | 10m 低端 | ✓ |
| 50 | 22.19 | 12m (24.89-24.99) | ✓ |
| 75 | 18.12 | 17m (18.068-18.168) | ✓ 完美对齐 |
| 110 | 14.96 | 15m (21.0-21.45) | 边界, 需核对 |
| 110 | 14.96 | — | — |
| 160 | 12.41 | — | WARC 边界 |
| 220 | 10.58 | 30m (10.1-10.15) | ✓ 完美对齐 |
| 350 | 8.39 | — | — |
| 510 | **6.94** | 40m (7.0-7.3) | ✓ 覆盖 |

**结论**: L2≈2.06μH + C=10-500pF 的组合**覆盖 6.94–35.07 MHz**。40m 到 10m 全部 WARC 频段**在线性范围内**，无需电感抽头。

### 2.4 最优 L 值敏感度分析

考察不同 L2 值对覆盖范围的影响:

| L2 (μH) | f_min @ 510pF (MHz) | f_max @ 20pF (MHz) | 40m 覆盖 | 10m 覆盖 |
|:-------:|:--------------------:|:-------------------:|:--------:|:--------:|
| 1.5 | 5.76 | 41.03 | ✓ | ✓ |
| **2.06** | **6.94** | **35.07** | ✓ | ✓ |
| 2.5 | 6.30 | 31.91 | 边界 | ✓ |
| 3.0 | 5.76 | 29.17 | ✓ | 边界 (28.0 MHz 需更小 C) |

> **最优区间**: L2 ∈ [1.8, 2.2] μH。当前 2.06 μH 处于区间中点 — **无需调整**。

## 3. Q 值与效率分析

### 3.1 空载 Q (磁芯本身)

```
Q_u = ωL / R_core_loss

Type 6 在 HF 的 Q 特性 (Micrometals 数据):
  @ 7 MHz:  Q_u ≈ 160
  @ 14 MHz: Q_u ≈ 150
  @ 28 MHz: Q_u ≈ 120

R_core_loss = 2πfL / Q_u
  @ 7 MHz:  R_loss = 2π×7e6×2.06e-6 / 160 ≈ 0.57 Ω
  @ 14 MHz: R_loss = 2π×14e6×2.06e-6 / 150 ≈ 1.21 Ω
  @ 28 MHz: R_loss = 2π×28e6×2.06e-6 / 120 ≈ 3.02 Ω
```

### 3.2 有效并联阻抗 (谐振时)

```
并联 LC 在谐振时呈现高阻抗:

Rp = Q_u × 2πfL = (2πfL)² / R_loss

  @ 7 MHz:  XL = 90.6 Ω → Rp = 160 × 90.6 ≈ 14,500 Ω
  @ 14 MHz: XL = 181.2 Ω → Rp = 150 × 181.2 ≈ 27,200 Ω
  @ 28 MHz: XL = 362.4 Ω → Rp = 120 × 362.4 ≈ 43,500 Ω
```

### 3.3 有载 Q 与匹配效率

```
天线端等效负载 = 2,450Ω (谐振 EFHW)

Q_loaded = R_ant / XL = 2,450 / XL

  @ 7 MHz:  Q_L = 2,450 / 90.6 ≈ 27.0
  @ 14 MHz: Q_L = 2,450 / 181.2 ≈ 13.5
  @ 28 MHz: Q_L = 2,450 / 362.4 ≈ 6.76

匹配效率 = Rp / (Rp + R_ant) = 1 / (1 + R_ant/Rp)

  @ 7 MHz:  η = 14,500 / (14,500 + 2,450) = 85.5%
  @ 14 MHz: η = 27,200 / (27,200 + 2,450) = 91.7%
  @ 28 MHz: η = 43,500 / (43,500 + 2,450) = 94.7%
```

> **发现**: 效率在低频端 (40m) 较低 (85.5%), 高频端 (10m) 极高 (94.7%)。原因是 XL 随频率上升而 Q_loaded 下降。**40m 的效率是设计的瓶颈**。

### 3.4 效率优化: 匝数比分析

改变匝数比 → 改变 Z_out → 改变 Q_loaded:

| N2 (次级匝) | 匝数比 | Z_out (Ω) | η@7MHz | η@14MHz | η@28MHz |
|:----------:|:-----:|:---------:|:------:|:-------:|:-------:|
| 12 | 6:1 | **36:1** → 1,800Ω | **88.4%** | 93.0% | 94.2% |
| 13 | 6.5:1 | 42.25:1 → 2,112Ω | 86.9% | 92.4% | 94.4% |
| **14** | **7:1** | **49:1 → 2,450Ω** | **85.5%** | 91.7% | 94.7% |
| 15 | 7.5:1 | 56.25:1 → 2,812Ω | 84.1% | 90.9% | 94.9% |
| 16 | 8:1 | 64:1 → 3,200Ω | 82.3% | 89.8% | 95.2% |

**关键洞察**:
- 低频段 (40m) 效率对匝数比**敏感**: 12 匝到 16 匝, 效率从 88.4% 降到 82.3% (差 6 个百分点)
- 高频段 (10m) 效率对匝数比**不敏感**: 始终 > 94%
- 当前 2:14 (7:1) 是**合理的折中** — 兼顾了 Z_out=2,450Ω (匹配典型 EFHW) 和效率

> **优化建议**: 如果实测 40m 效率过低, 可尝试 **2:13 (6.5:1 → Z_out=2,112Ω)**, 效率提升 1.4 个百分点。但需确认天线实际阻抗是否匹配 2,112Ω。

## 4. 阻抗变换精度分析

### 4.1 理想变压器 vs 实际耦合

```
理想: Z_out,ideal = Z_in × n² = 50 × 49 = 2,450 Ω

实际 (含耦合系数 k ≈ 0.92-0.96):
  Z_out,actual = Z_in × (n × k)²
  
  k=0.92: Z_out = 50 × (7 × 0.92)² = 50 × 41.5 ≈ 2,075 Ω  (误差 -15%)
  k=0.95: Z_out = 50 × (7 × 0.95)² = 50 × 44.2 ≈ 2,210 Ω  (误差 -10%)
  k=0.98: Z_out = 50 × (7 × 0.98)² = 50 × 47.0 ≈ 2,350 Ω  (误差 -4%)
```

耦合系数对实际阻抗影响显著。**初级 2 匝紧密绕在次级绕组内的布局**可最大化 k。

### 4.2 SWR 对阻抗偏差的敏感度

```
假设天线实际阻抗 Z_ant = 2,450Ω, 变压器输出 Z_out 有偏差:

Z_out (Ω)  反射系数 |Γ|  SWR
2,450       0          1.00:1  (理想)
2,200       0.053      1.11:1
2,100       0.077      1.17:1
1,800       0.153      1.36:1
1,500       0.240      1.63:1
```

> 变压器输出在 2,100-2,800Ω 范围内时 SWR<1.2:1 — 足够好。**不需要过分追求精确 2,450Ω**。

## 5. B_peak 饱和校核 (精确计算)

### 5.1 40m 最严条件 (100W PEP)

```
V_in = √(P × Z_in) × √2 = √(100 × 50) × √2 = 70.7 × √2 ≈ 100 V_peak (初级)

V_sec = V_in × n = 100 × 7 = 700 V_peak (次级)

磁通密度:
B_peak = V_sec / (4.44 × f × N2 × A_e)
       = 700 / (4.44 × 7.1e6 × 14 × 1.27e-4)
       = 700 / (4.44 × 7.1e6 × 1.778e-3)
       = 700 / 5.60e4
       ≈ 12.5 mT
```

**120W PEP (含余量)**:
```
V_sec = √(120 × 50) × √2 × 7 ≈ 109.5 × 7 ≈ 767 V_peak
B_peak = 767 / 5.60e4 ≈ 13.7 mT
```

| 功率 | B_peak (mT) | 裕度 (vs 600mT) |
|:----:|:-----------:|:---------------:|
| 50W | 8.8 | 68× |
| 100W | 12.5 | **48×** |
| 120W | 13.7 | 44× |
| 200W | 17.7 | 34× |

> **结论**: T200-6 单只在 100W 下 48× 裕度，**200W 仍有余量 34×**。单只磁芯完全足够，双叠无必要。

## 6. 电容峰值电压

### 6.1 谐振时电容电压

```
V_cap = Q_loaded × V_sec

Q_loaded @ 各频段 (见 §3.3):

  @ 7 MHz (40m):  V_cap = 27.0 × 700 = 18,900 V_peak  ← ⚠️ 极高!
  @ 14 MHz (20m): V_cap = 13.5 × 700 = 9,450 V_peak
  @ 28 MHz (10m): V_cap = 6.76 × 700 = 4,732 V_peak
```

**但这个值是理论最大值!** 实际中 Q_loaded 会被天线本身的损耗和传输线损耗大幅降低:

```
实际 Q_eff = Q_loaded × (R_ant / (R_ant + R_loss_total))
  R_loss_total 包含: 天线导体损耗 + 地面损耗 + 传输线损耗 + 接触电阻

若 R_loss_total ≈ 200-500Ω (典型 EFHW):
  Q_eff @ 7 MHz ≈ 27 × (2450/(2450+300)) ≈ 24
  V_cap_actual ≈ 24 × 700 ≈ 16,800 V_peak

若天线不在完美谐振点 (SWR>1.0, Q_loaded 进一步降低):
  V_cap 可能低至 5,000-10,000 V_peak
```

### 6.2 空气可变电容耐压评估

```
典型空气可变电容 (广播收音机用):
  片间距: 0.3-0.5mm
  耐压: 500-1,500V (随片间距增加, 容量减少)
  
  10pF 位置 (片间距最大): 耐压 ≈ 1,500V
  500pF 位置 (片间距最小): 耐压 ≈ 300-500V

⚠️ 在最坏条件下 (40m, 完美匹配, 100W):
  电容电压可能达到 5-17 kV → 远超 1,500V 额定!
```

**这是本设计最重要的风险!** 解决方案:

| 方案 | 优点 | 缺点 |
|------|------|------|
| **限制 40m 功率到 50W** | 简单 (固件限制) | 牺牲功率 |
| **使用发射机级空气可变电容** (片间距 ≥1mm) | 耐压 ≥3kV | 体积大, 难找 |
| **双电容串联分压** | 每个承担一半电压 | 需要两个伺服同步 |
| **并联放电器 (spark gap) 保护** | 简单低成本 | 可能反复打火 |

> **推荐**: 第一优先级是升级为发射机级空气可变电容 (工作耐压≥5kV或片距≥1.5mm) 并保证RF热端物理间距。不要把90V/200V低压GDT默认并接在RF热端到地, 否则可能在正常RF电压下误放电。保护件仅预留可调空气火花隙或3-5kV低电容GDT, 台架确认不误触发后再装。固件/操作上仍应限制40m调谐功率到≤5W。

## 7. 最优参数建议

基于以上全部分析:

| 参数 | 原设计 | 优化建议 | 理由 |
|------|:------:|:--------:|------|
| T200-6 数量 | 1 | **1** (不变) | B_peak 48× 裕度, 双叠过度设计 |
| 初级匝数 | 2 | **2** (不变) | 标准做法 |
| 次级匝数 | 14 | **13-14** (灵活) | 如果 40m 效率 <80% 则降为 13T |
| L2 (μH) | 2.06 | **1.8-2.2** (不变) | 最优区间中心 |
| 可变电容 C_max | 500 pF | **500 pF** (不变) | 下限 f_min=6.94 MHz 完美对齐 40m |
| 可变电容耐压 | ≥1 kV | **≥5 kV 或片距≥1.5mm** (升级!) | 40m 谐振电压可达数kV, 普通收音机电容风险过高 |
| 电容保护 | 无 | **预留HV火花隙/3-5kV低电容GDT, 默认DNP** | 浪涌/误调保护; 不使用低压GDT |
| 40m 调谐功率 | 5W | **5W** (已够) | 通过 MRRC 限制 fwd_pwr, 不需要变 |
| 耦合系数 k | ≥0.90 | **≥0.95** (绕法优化) | 初级紧贴次级内层绕制 |

### 最关键的优化动作:

1. **评估可变电容耐压** — 如果是普通收音机电容 (片间距 <0.5mm), 必须加 GDT 保护或升级
2. **优化绕法** — 初级 2 匝紧贴次级, 提高耦合系数到 0.95+ 
3. **实测 40m 效率** — 如果 <80%, 尝试降低次级匝数到 13T 或 12T

## 8. Python 谐振曲线计算

以下 Python 脚本可生成频率响应数据 (替代 ngspice 扫频):

```python
#!/usr/bin/env python3
"""
V3.0 Fuchs ATU — LC 谐振特性计算
替代 ngspice AC 扫频
"""

import math

# ===== 设计参数 =====
L2 = 2.058e-6       # 次级电感 (H)
C_stray = 10e-12    # 杂散电容 (F)
Z_in = 50           # 输入阻抗
n = 7               # 匝数比 (14/2)
k = 0.93            # 耦合系数
P_in = 100          # 输入功率 (W)

# ===== Type 6 Q 值 vs 频率 (Micrometals 数据拟合) =====
def Q_u(f):
    """空载 Q 值 @ 频率 f (Hz)"""
    if f < 10e6:
        return 160
    elif f < 20e6:
        return 155 - (f - 10e6) * 1e-6  # 线性插值
    else:
        return max(120, 145 - (f - 20e6) * 1.5e-6)

# ===== 计算谐振频率 =====
def f_res(C):
    return 1.0 / (2 * math.pi * math.sqrt(L2 * C))

# ===== 频率扫描 =====
print(f"{'Cap(pF)':>8s} {'f_res(MHz)':>12s} {'XL(Ω)':>8s} {'Q_u':>6s} {'Q_L':>6s} {'Eff(%)':>8s} {'B_peak(mT)':>12s}")
print("-" * 72)

for C_pF in range(20, 520, 30):
    C = C_pF * 1e-12 + C_stray
    f = f_res(C)
    
    if f > 30e6 or f < 6.5e6:
        continue  # 超出频段范围
    
    XL = 2 * math.pi * f * L2
    Qu = Q_u(f)
    
    Z_out_ideal = Z_in * (n * k)**2
    Q_loaded = Z_out_ideal / XL if XL > 0 else 50
    efficiency = (1 - Q_loaded / Qu) * 100 if Qu > 0 else 0
    
    # B_peak
    V_in_peak = math.sqrt(P_in * Z_in) * math.sqrt(2)
    V_sec = V_in_peak * n
    A_e = 1.27e-4
    N2 = 14
    B_peak = V_sec / (4.44 * f * N2 * A_e) * 1000  # mT
    
    print(f"{C_pF:8.0f} {f/1e6:12.2f} {XL:8.1f} {Qu:6.0f} {Q_loaded:6.1f} {efficiency:8.1f} {B_peak:12.1f}")

# ===== 最优电容 vs 频率反推 =====
print(f"\n{'Freq(MHz)':>10s} {'Optimal C(pF)':>15s}")
print("-" * 30)
for f_target in [7.100, 10.125, 14.200, 18.118, 21.200, 24.940, 28.500]:
    C_opt = 1.0 / ((2 * math.pi * f_target * 1e6)**2 * L2) * 1e12 - C_stray * 1e12
    print(f"{f_target:10.3f} {C_opt:15.1f}")
```
