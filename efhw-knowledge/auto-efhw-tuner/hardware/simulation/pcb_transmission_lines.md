# PCB 传输线参数 — 仿真级分析
# =================================
# 用于 KiCad 的 PCB 传输线计算工具 (TX-Line) 或手动解析计算
# 目标: 确保 50Ω 主线匹配, 高压轨阻抗受控, 无意外谐振

## 1. PCB 叠层参数

```
Layer Stack (双面板, FR4, 1.6mm):

  ┌──────────────────────────────────────┐  ← 顶层 F.Cu (35µm 铜)
  │  信号走线, MCU 控制, 电源轨          │
  ├──────────────────────────────────────┤
  │  FR4 介质层 (Er≈4.5 @ 14MHz)         │  1.6mm
  │  Tanδ ≈ 0.02 @ 14MHz                │
  ├──────────────────────────────────────┤
  │  底层 B.Cu (35µm 铜)                 │
  │  完整数字地平面 (单区, 无分割)       │
  └──────────────────────────────────────┘
```

| 参数 | 值 | 备注 |
|------|-----|------|
| 板材 | FR-4, Tg≈135°C | 标准双面板 |
| 介电常数 Er | 4.5 @ 14 MHz | 典型 FR4 |
| 损耗角 Tanδ | 0.020 @ 14 MHz | 随频率增加 |
| 铜厚 | 1 oz/ft² (35 µm) | 标准 |
| 板厚 | 1.6 mm | 标准 1/16" |
| 阻焊厚度 | ~25 µm | 绿色阻焊 |

## 2. 关键传输线分析

### 2.1 RF 主线: SO-239 → T200-6 初级 (50Ω)

**微带线 (Microstrip)**:
- 顶层信号走线, 底层完整地平面
- 目标阻抗: 50 Ω ± 10%

```
计算 (TX-Line 或 Wheeler 公式):

  线宽 (W):     2.85 mm  (50Ω)
  铜厚 (t):     0.035 mm
  介质厚度 (h): 1.6 mm
  Er:           4.5

  Z₀:          50.1 Ω  ✓
  ε_eff:       3.45
  v_phase:      1.61 × 10⁸ m/s (0.54c)
  波长 @14MHz:  ~11.5 m (空间) → ~6.2 m (PCB上)
  λ/10:         62 cm → PCB 主线 << λ/10 → 无需阻抗控制!
```

> **结论**: RF 主线在 PCB 上的长度 (约 5-8 cm) 远小于 λ/10 (62 cm @ 14 MHz)。在 HF 频段, **PCB 走线不作为传输线工作**, 而是集总元件。50Ω 匹配由外部保证。

### 2.2 HV 飞线: T200-6 次级热端 → 天线端子 (高压, 板外点对点)

**特性**:
- RF 高压路径不经过 PCB，采用 1.5mm² 铜线点对点飞线
- 工作电压: ≤ 3KV peak
- C_stray 仅 ~4pF (无 PCB 寄生电容)

```
安全分析:

  爬电路径: 空气介质 + PTFE 绝缘
  C_stray ≈ 4 pF (实测/等效)
  
  在 30 MHz, 4pF 的阻抗:
  Xc = 1/(2π × 30MHz × 4pF) ≈ 1,326 Ω
  → 远优于 PCB 走线的 ~6pF → 保障 10m 波段覆盖 ✓
```

### 2.3 伺服供电轨 (高电流)

```
VCC_6V → IRF9540 → VCC_SERVO → HDR_SERVO Pin2

走线宽度: 2.0mm (1oz 铜 ≈ 3A 容量)
最大电流: 3A (伺服堵转瞬态)
PCB 上长度: ~40mm
```

### 2.4 过孔 (Via) 建模

| 过孔用途 | 孔直径 | 焊盘直径 | 数量 | 寄生电感 (每个) |
|---------|--------|---------|------|---------------|
| MCU VSS → 底层地 | 0.5mm | 1.0mm | 2/Pin | ~1.0 nH |
| 电源 IC GND → 底层地 | 0.6mm | 1.2mm | ≥2/IC | ~0.8 nH |

```
过孔寄生电感 (近似公式):
  L_via ≈ 5.08 × h × [ln(4h/d) + 1]  (nH, 尺寸单位 inch)
  
  对于 h=1.6mm(0.063"), d=0.6mm(0.024"):
  L_via ≈ 5.08 × 0.063 × [ln(4×0.063/0.024) + 1]
        ≈ 5.08 × 0.063 × [ln(10.5) + 1]
        ≈ 5.08 × 0.063 × 3.35
        ≈ 1.07 nH  (每个过孔)

  3 个过孔并联: 1.07/3 ≈ 0.36 nH  ← 对 HF 频段可忽略
  
  @ 30 MHz, XL = 2π × 30MHz × 0.36nH ≈ 0.068 Ω  ✓
```

## 3. 传输线工具使用 (KiCad / TX-Line)

### 3.1 KiCad 内置传输线计算器

```
PCB Editor → Tools → Transmission Line Calculator

推荐设置:
  Type:       Microstrip
  Er:         4.5
  TanD:       0.02
  Height:     1.6  mm
  Thickness:  0.035 mm
  Frequency:  30 MHz

  目标 Z₀:    50 Ω → 线宽 = 2.85 mm
  目标 Z₀:    75 Ω → 线宽 = 0.98 mm
  目标 Z₀:   100 Ω → 线宽 = 0.38 mm

  (这些仅在走线长度 > λ/10 时才有意义)
```

## 4. 完整性检查清单

| 检查项 | 方法 | 通过标准 |
|--------|------|---------|
| RF 主线回波损耗 | VNA S11 @ M座 | < -20 dB @ 1-30 MHz |
| HV_BUS 对地电容 | LCR 表 @ 100 kHz | < 10 pF |
| 过孔总电感 (电容冷端) | 只对极高频率 (< 100 MHz) 有意义 | < 5 nH |
| 控制线串扰 | 相邻线接 50Ω → 另一根测耦合 | < -40 dB @ DC-1 MHz |
| 地平面连续性 | 目视 + 万用表蜂鸣 | A区底层无断裂 ✓ |

---

> HF 频段 (1.8-30 MHz) PCB 走线关键结论:
> 1. 走线长度 << λ/10, 传输线效应可忽略
> 2. 阻抗匹配由集总元件 (T200-6 变压器 + 可变电容) 实现
> 3. RF 高压路径不经过 PCB (点对点飞线), C_stray 仅 ~4pF
> 4. 主要关注: 伺服供电轨 IR 压降、地平面完整性
