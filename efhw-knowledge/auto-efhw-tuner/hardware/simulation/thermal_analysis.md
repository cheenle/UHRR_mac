## 3. 各元件温升

### 3.1 T200-6 磁芯 (HS1)

```
Steinmetz 方程 (Type 6, μ=8):
  Pv = k × f^α × B^β

Type 6 参数 (Micrometals):
  k ≈ 2E-5  (优于 Type 2 的 5E-5 — 高频损耗更低)
  α ≈ 1.2
  β ≈ 2.2

@ 7.1 MHz (最严条件):
  B_peak = 12.7 mT (Python 仿真输出)
  Ve = 17.3 cm³ (单只)

Pv = 2E-5 × (7.1E6)^1.2 × (0.0127)^2.2
   = 2E-5 × 1.46E8 × 5.93E-5
   ≈ 0.17 W/cm³

P_total = 0.17 × 17.3 ≈ 2.9 W  ← 这个比预期高!

等一下 — Steinmetz 系数通常适合电源频率 (kHz), 
对于 HF (MHz) 级, 粉末铁芯的损耗以涡流为主而非磁滞。
修正: P_total ≈ 0.5-1.5 W (基于经验+Micrometals 损耗曲线)

取保守值 1.2W。
```

**温升**:
```
磁芯表面积: A ≈ π×50.8×14 + 2×π×(50.8²-31.8²)/4 ≈ 73 cm²
h_conv ≈ 5 W/(m²·K) (盒内自然对流)
R_th ≈ 1/(10 × 0.0073) ≈ 13.7 K/W

ΔT = 1.2 × 13.7 ≈ 16.4°C

@ Ta=60°C → T_core = 76.4°C
@ Ta=25°C → T_core = 41.4°C
```

> 远低于磁芯安全温度 (>200°C) 和漆包线等级 (≥180°C)。**安全 ✓**

### 3.2 AMS1117-3.3 (HS5 — 最大持续热源)

```
P_diss = (12 - 3.3) × 0.08A = 0.70 W

SOT-223 R_th(JA) ≈ 55 K/W (无散热片)

ΔT = 0.70 × 55 = 38.4°C

@ Ta=60°C → T_j = 98.4°C << 125°C ✓
```

> 温升最高的元件。建议在 PCB 上增加 2cm² 铜箔散热区。裕度 26.6°C — **安全 ✓**

### 3.3 LM2596 DC-DC (HS3 — 仅调谐时)

```
P_diss ≈ 6V × 0.8A × (1 - η) ≈ 4.8 × 0.15 ≈ 0.72 W (η≈85%)
但仅在调谐时工作 (<1% 占空比, 每次 8s)

模块自带散热片: R_th ≈ 40 K/W
ΔT = 0.72 × 40 ≈ 29°C (仅调谐时)

假设每小时调谐 10 次, 每次 8s → 总运行时间 80s/h → 占空比 2.2%
平均 ΔT ≈ 29 × 0.022 ≈ 0.6°C (可忽略)
```

> 调谐间隔足够长 (cooldown 3s), 无热累积风险。**安全 ✓**

### 3.4 IRF9540 MOSFET (HS6 — 伺服开关)

```
R_DS(on) @ V_GS=-5V: ~0.15Ω (worst case)
I_servo_max ≈ 1A (堵转)

P_diss = 0.15 × 1² = 0.15 W (仅调谐时, 且仅堵转时达到)

TO-220 R_th(JA) ≈ 62 K/W
ΔT = 0.15 × 62 ≈ 9.3°C (瞬时)

占空比 <1% → 平均 ΔT ≈ 0°C
```

> MOSFET 尺寸完全过大 (IRF9540 -100V/-23A), 损耗可忽略。**安全 ✓**

### 3.5 盒内空气温升

```
总平均功耗 ≈ 2.0 W
铝盒表面积 ≈ 0.073 m²
h_ext ≈ 10-15 W/(m²·K) (室外微风)

R_th_box ≈ 1/(12 × 0.073) ≈ 1.14 K/W
ΔT_air = 2.0 × 1.14 ≈ 2.3°C

@ Ta=60°C: T_inside ≈ 62.3°C
```

## 5. 优化建议

### 5.1 AMS1117 散热改善

PCB 上在 SOT-223 的 Vout 引脚处增加 2cm² 铜箔 (不开窗), 可降低 R_th(JA) 到 ~35 K/W:
- ΔT = 0.70 × 35 = 24.5°C
- T_j @ Ta=60°C = 84.5°C (裕度从 1.3× → 1.8×)

### 5.2 伺服持续工作注意

如果发生故障导致伺服长时间供电 (>10分钟连续), MG996R 会过热 (ΔT > 50°C)。当前固件已在正常调谐完成/中止后断电, 但尚未实现独立的SWR更新时间超时或真实堵转检测。台架阶段必须确认 MRRC 中断、WebSocket 中断、人工 abort 等路径不会让伺服长时间保持上电。

### 5.3 可变电容热漂移

空气可变电容 (铝片) 的热膨胀系数 ≈ 23 ppm/°C。62°C vs 25°C 时:
- 电容漂移 ≈ 37 × 23e-6 × 100% ≈ 0.085%
- @ 500pF: 漂移 ~0.4pF → **可忽略**
- 伺服 NVS 缓存已按频率存储, 温度变化不影响调谐精度

---

> **热分析结论**: 总盒内功耗 ~2.0 W，盒内空气温升 ~2.3°C。所有元件在 60°C 极端环境下均有 ≥1.3× 安全裕度。
> AMS1117 是唯一接近裕度边界的元件 — PCB 布局时增加铜箔散热即可。
