## 2. 叠层结构

```
Layer Stack (双面板):
  F.Silkscreen  → 顶层丝印
  F.Mask        → 阻焊 (绿色)
  F.Cu (35µm)   → 信号 + 电源
  FR-4 (1.6mm)  → Er≈4.5
  B.Cu (35µm)   → 完整地平面 (无分割)
  B.Mask        → 阻焊
```

低压控制区使用连续地平面。RF高压谐振区不在主PCB上实现, 通过底板点对点硬线连接并与控制PCB保持机械隔离。

### 2.1 物理分区原则

| 区域 | 位置 | 约束 |
|------|------|------|
| RF/HV 谐振区 | 铝壳右侧或上半区底板 | T200-6、C_var、ANT端子之间用短而圆滑的硬铜线连接; 与PCB、ESP天线、低压线束保持 ≥15mm 空气距离 |
| 低压控制区 | PCB整体 | 只承载DC、电源、MCU、PWM、ADC; 不允许RF二次侧高压网络进入 |
| 伺服/齿轮区 | C_var轴侧 | 伺服固定孔与电容轴同轴; 齿轮啮合留0.1-0.2mm侧隙; 伺服线束避开RF热端 |
| WiFi天线区 | ESP32-S3天线前方 | 天线正前方和上下方 ≥15mm 禁铜、禁金属支架; 铝壳开窗或将天线端靠近塑料防水窗 |
| 电源噪声区 | LM2596附近 | 屏蔽电感模块靠板边, 输入/输出环路最短; 与ADC分压和ESP天线保持距离 |

## 4. 关键布线

### 4.1 伺服电源路径 (高电流)

```
VCC_6V (LM2596 OUT) ── 2.0mm宽走线 ──→ Q1(IRF9540) Source
Q1 Drain ── 2.0mm宽走线 ──→ HDR_SERVO Pin2 (VCC_SERVO)
HDR_SERVO Pin1 → GND (地平面)
Q1 Gate ←── Q2(2N2222A) Collector + 10kΩ pull-up 到 VCC_6V
Q2 Base ←── 1kΩ ←── GPIO2 (SERVO_PWR_EN)
```

伺服供电轨最大 3A (堵转)。走线宽度 ≥2.0mm (1oz铜 ≈ 3A 容量)。PCB 上 6V 轨加 100µF 电解 + 100nF 陶瓷旁路, HDR_SERVO旁再加100µF低ESR钽/固态电解。采用常见NPN下拉P-MOS栅极时, 逻辑为 GPIO2 HIGH=伺服上电, GPIO2 LOW=伺服断电; 固件命名应使用 `SERVO_PWR_EN`, 避免与 `POWER_CUT` 反义混淆。

### 4.2 伺服信号

```
GPIO1 ── 0.3mm走线 ── HDR_SERVO Pin3
```

PWM 50Hz 信号，低电流。常规走线即可。

### 4.3 Bias-T 电压监测

```
VCC_12V ── 0.3mm走线 ── R_bias_div1(47kΩ) ──┬── GPIO5
                                              │
                                        R_bias_div2(10kΩ)
                                              │
                                             GND
```

高阻抗分压 (总 57kΩ)，走线远离伺服 PWM、LM2596开关节点和大电流区域，100nF 电容紧靠 GPIO5。ADC走线两侧优先用地铜护线, 不跨越6V伺服回流路径。

### 4.4 天线/TRX 接线 (板外飞线)

```
SO-239 芯线 → (最短连接) → C_block1/2 → T200-6 初级2匝热端
T200-6 初级冷端 → (飞线) → SO-239 外壳 (GND)
T200-6 次级热端 → (硬铜线) → 可变电容定片 → (硬铜线) → ANT M5 端子
T200-6 次级冷端 / 可变电容动片 → 铝壳单点地
```

RF二次侧可出现数kV峰值。所有二次侧热端连接必须使用1.5mm²以上硬铜线或PTFE高压线, 端点打磨圆滑, 不留尖角焊锡, 与PCB/低压线束/铝壳非接地点保持 ≥15mm 空气距离。普通90V或200V GDT不得直接并在RF热端到地作为默认装配, 否则会在正常RF电压下误触发; 如需保护, 仅预留可调空气火花隙或3-5kV低电容器件并在台架确认不误放电。

### 4.5 壳体内推荐布局

```
俯视, 160×110×70mm 铝壳示意:

SO-239面板
┌────────────────────────────────────────┐
│ RF输入/C_block  T200-6        ANT M5   │  ← RF/HV区, 短硬线, 远离PCB
│      │           │││           │       │
│      └────────── C_var ────────┘       │
│                    │                   │
│                齿轮组 + MG996R         │
│                                        │
│ PCB低压控制板 140×50                  │
│ [电源] [ESP32天线朝塑料窗/壳体边缘]    │
└────────────────────────────────────────┘
```

机械约束:

| 项目 | 要求 |
|------|------|
| C_var | 发射机级空气电容, 片距≥1.5mm; 用绝缘支柱固定, 定片热端远离壳体 |
| T200-6 | 尼龙扎带或3D打印支架固定, 磁环下垫PTFE/尼龙片, 不直接磨铝底板 |
| MG996R | 金属支架固定到壳体底板, 伺服轴与电容轴平行; 齿轮护罩避免线束卷入 |
| RF热端 | 保持短、直、圆滑; 避开LED/蜂鸣器/ESP32天线/USB口 |
| 地点 | SO-239法兰、T200初级冷端、C_var动片、PCB GND在SO-239附近单点汇合 |
| 防潮 | 壳体底部放干燥剂; 透气孔加PTFE防水透气膜, 不开裸孔 |

## 6. 制造说明

| 参数 | 规格 |
|------|------|
| 板材 | FR-4, Tg≥135°C, 1.6mm |
| 铜厚 | 1oz (35µm) |
| 表面处理 | ENIG (推荐) 或 HASL |
| 阻焊 | 绿色 |
| 最小钻孔 | 0.3mm |
| 测试 | 飞针 100% netlist |

### 装配顺序

```
1. 回流焊: ESP32-S3模组, SOT-223 AMS1117, 0805/1206 R/C
2. 手工焊: TO-220 LM2940, TO-220 IRF9540, TO-92 2N2222A ×2
3. 手工焊: DO-41 1N4007, LM2596模块, 电解电容
4. 手工焊: 伺服排针, 接线端子, LED, 蜂鸣器
5. 入壳: 尼龙柱支撑 PCB → M座拧紧, 确认ESP32天线净空
6. 安装板外件: T200-6 尼龙扎带/支架固定 → 发射机级可变电容+伺服 螺丝固定 → 齿轮耦合
7. RF 硬线: 按 §4.4 点对点连接 (1.5mm² 铜线/PTFE高压线), 保持HV间距
8. ESP32-S3 通过 USB 烧录初版固件 → WiFi 配置
9. 三防漆喷涂 (仅 PCB 面, 避开连接器和散热器)
```
