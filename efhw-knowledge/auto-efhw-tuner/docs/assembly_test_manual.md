# EFHW Fuchs ATU V3.0 — 装配、测试与故障排查手册

> **版本**: V3.0 (2026-06-09) — ESP32-S3 + T200-6 + MG996R
> **适用**: 具备基础焊接与射频测量能力的业余无线电爱好者

---

## 2. 装配顺序

### 2.1 PCB 焊接 (140×50mm)

```
1. 回流焊/热风枪:
   - ESP32-S3-WROOM-1 (SMD 模组)
   - AMS1117-3.3 (SOT-223)
   - 0805/1206 阻容件 (全部 ~25 件)

2. 手工焊 (粗脚):
   - LM2940CT-12 (TO-220) — 注意散热片方向
   - IRF9540 (TO-220) — 伺服供电 P-MOSFET
   - 2N2222A ×2 (TO-92) — MOSFET 栅极驱动 + 蜂鸣器驱动
   - 1N4007 (DO-41) — 防反二极管

3. 手工焊 (模块/接插件):
   - 屏蔽型 LM2596/MP1584 DC-DC 模块 (插入排母或直接焊)
   - 电解电容 ×4 (47µF/100µF, 注意极性!)
   - 3-pin 伺服排针 (板底边缘)
   - LED (3mm) + 蜂鸣器 (5V有源)
   - 接线端子 (电源输入, 可选)

4. 不焊在 PCB 上的:
   - SO-239 (面板安装)
   - T200-6 (尼龙扎带固定在底板上)
   - 可变电容 (螺丝固定在底板上)
   - MG996R (螺丝固定在底板上)
   - HV_GAP/高压GDT预留件 (默认不装; 台架确认后再评估)
```

### 2.2 磁环绕制

```
1. 材料: T200-6 ×1, 0.8mm 漆包线 ~0.5m, 0.5mm 漆包线 ~2m, PTFE套管

2. 初级 (inner winding):
   0.8mm 漆包线, 绕 2 匝, 引线套PTFE管
   首尾预留 5cm 剥漆上锡

3. 次级 (outer winding):
   0.5mm 漆包线, 绕 14 匝
   均匀分布在磁环上 (约 25° 间距), 不要密绕
   首尾预留 5cm 剥漆上锡

4. 固定: T200下垫PTFE/尼龙片, 尼龙扎带穿过中心孔固定在底板上

5. 验证: LCR 表测初级电感 ≈ 2.06µH ±10%
```

### 2.3 可变电容 + 伺服安装

```
1. 发射机级可变电容固定在底板上 (M3 螺丝/绝缘支柱), 确认片距≥1.5mm或工作耐压≥5kV
2. MG996R 伺服固定在底板上 (M3 螺丝)
3. 齿轮组: 伺服输出轴 → 减速齿轮 → 电容轴
   - 伺服 0° (500µs) = 电容最小值 ≈ 10pF
   - 伺服 180° (2500µs) = 电容最大值 ≈ 500pF
4. 验证: 手动转动齿轮, 确认无卡涩, 电容片不碰片; 齿轮侧隙约0.1-0.2mm
5. 用 LCR 表在两个端点确认电容值
```

### 2.4 RF 接线 (飞线)

```
SO-239 芯线 → (最短硬线) → C_block ×2 (2kV C0G/NP0或银云母)
C_block 冷端 → (1.5mm²铜线/PTFE高压线) → T200-6 初级 2匝 热端
T200-6 初级 2匝 冷端 → (1.5mm²铜线) → SO-239 外壳 (GND)

T200-6 次级 14匝 热端 → (1.5mm²硬铜线/PTFE高压线) → 可变电容 定片 → M5 ANT 端子
T200-6 次级 14匝 冷端 → (1.5mm²铜线) → GND 汇流点
可变电容 动片 → GND 汇流点

HV_GAP: 默认DNP; 如安装, 使用可调空气火花隙或3-5kV低电容GDT, 确认正常调谐不误放电
R_bleed (2.2MΩ): 并联在 ANT 端子-GND 之间
```

RF二次侧热端可能出现数kV峰值。所有焊点必须圆滑, 不留尖角锡峰; RF热端与PCB、伺服线、低压线束和非接地点铝壳保持 ≥15mm 空气距离。禁止把90V/200V低压GDT默认焊在RF热端到地。

### 2.5 入壳

```
1. 尼龙柱支撑 PCB 固定到铝壳底板 (M3 螺丝)
2. SO-239 法兰拧紧到面板开孔
3. M5 天线端子 + 地网端子 拧紧到面板
4. 伺服 3-pin 线插入 HDR_SERVO 排针
5. 整理所有飞线, RF热端线束与低压线束分开扎带固定
6. 三防漆喷涂 (仅 PCB 面, 避开连接器/散热器)
7. 干燥剂包放入, 合盖, 拧紧螺丝
8. PTFE 透气膜覆盖底部呼吸孔
```

## 3. PCB 上电前检查 (无 DC)

| 检查 | 方法 | 预期 |
|------|------|------|
| 电源短路 | 万用表 12V 输入对 GND 测电阻 | > 1kΩ |
| 3.3V 短路 | 万用表 3.3V 轨对 GND 测电阻 | > 500Ω |
| 6V 短路 | 万用表 6V 轨 (LM2596 OUT) 对 GND | > 200Ω |
| 伺服 VCC 隔离 | Q1 (IRF9540) 未导通时 HDR_SERVO Pin2 对 6V 断路 | 不通 |
| MOSFET 栅极 | GPIO2 对 Q2 Base (经 1kΩ) 导通 | < 2Ω |

## 4. 首次上电

```
1. USB-C 连接电脑, 暂不接 Bias-T 供电
2. 烧录固件: idf.py flash
3. 打开串口: idf.py monitor (115200bps)
4. 观察启动日志:
   === Fuchs ATU V3.0 (ESP32-S3) ===
   Firmware built: Jun  9 2026
   servo: Initialized: PWM 50Hz, pin=1, power_en=2
   nvs_cache: Tune cache initialized (nvs_tune partition)
   health: Health monitor initialized (ADC + temp sensor)
   main: All tasks started.

5. 蜂鸣器响 1 声
6. LED: 因无 WiFi 配置, 预期 LED 快闪 (断连指示)
```

## 5. WiFi 配置

```
1. 通过串口发送 WiFi 凭据 (一次性配置):
   esp_wifi_set_config(WIFI_IF_STA, &wifi_config)
   ssid = "你的WiFi名"
   password = "你的WiFi密码"

2. 或编译前修改 sdkconfig.defaults:
   CONFIG_ESP_WIFI_SSID="你的WiFi名"
   CONFIG_ESP_WIFI_PASSWORD="你的WiFi密码"

3. 连接成功: 日志显示 WiFi connected, IP: xxx
4. LED: 由快闪变为常亮 (WS 连接后)
```

## 6. 功能测试

### 6.1 伺服测试 (无 RF)

| 测试 | 命令 | 预期 |
|------|------|------|
| 归零 | `{"cmd":"set_bypass"}` | 伺服转到 0°, 然后断电 (VCC_SERVO=0V) |
| 中位 | 手动发 servo_set_angle(90°) | 伺服转到 90° |
| 满量程 | 0° → 180° → 0° 往返 | 无卡涩, 无异响 |

### 6.2 Bias-T 供电测试

```
1. 连接 Bias-T 电源: 13.8V DC 注入同轴
2. 串口日志: health_mon 显示 Bias-V=13.8V (正常范围)
3. 万用表验证:
   - VCC_12V = 12.0V ±0.5V
   - VCC_6V = 6.0V ±0.3V (仅调谐时, 调谐结束 = 0V)
   - VCC_3V3 = 3.30V ±0.1V
```

### 6.3 调谐测试 (5W CW)

```
1. 电台设置: 14.200MHz, USB, 5W 输出
2. ATR1000 确认在线: MRRC 显示 SWR 读数
3. MRRC 发送: {"cmd":"tune_start","freq_hz":14200000,"swr":<当前SWR>,"fwd_pwr_w":5}
4. 观察:
   - 伺服开始摆动 (粗扫37个采样点, 0°→180° @5°)
   - 如果 SWR > 1.5, 细扫 30 步 (~3s)
   - 锁定后伺服断电
5. 最终 SWR < 2.0 (首次), SWR < 1.5 (同频段再次)
6. NVS 缓存写入确认: 日志显示 "Saved: 14200kHz → pos=XX"
```

## 8. 故障排查

| 症状 | 可能原因 | 排查 |
|------|---------|------|
| 上电无反应 (无 LED/蜂鸣) | 3.3V 轨故障 | 测 AMS1117 输出 = 3.3V? |
| LED 持续快闪 | WiFi 无法连接 | 检查 SSID/密码, 测 RSSI |
| WebSocket 连不上 | MRRC 未启动或 IP 错误 | ping MRRC, 检查 /atu 端点 |
| 伺服不动作 | 6V 轨断开 / MOSFET 损坏 | 测 VCC_SERVO 调谐时 = 6V? 测 IRF9540 导通? |
| 伺服抖动但不转 | 堵转 / 齿轮卡死 | 手动转动齿轮, 检查润滑 |
| SWR 随伺服变化不大 | 电容未接入 / 天线不谐振 | 检查 T200-6 接线, 检查天线长度 |
| 调谐后 SWR > 2.0 | 天线不匹配 / 电容范围不够 | 验证 C 范围 10-500pF, 验证 L≈2.0µH |
| 全部 SWR > 3.0 | 隔直电容断路 / 接线错误 | 逐点检查 RF 飞线连通性 |
| 调谐中掉电 | Bias-T 电压跌落 | 检查 Bias-T 电源容量 (>300mA) |
| ESP32-S3 反复重启 | 电源不稳 / WDT 触发 | 检查 3.3V 纹波, 查看复位原因 |

## 9. 维护周期

| 周期 | 操作 |
|------|------|
| 每次操作前 | 检查 SWR 正常 (MRRC 面板) |
| 每月 | 齿轮外观检查, 听有无异响 |
| 每 6 个月 | 开壳: 齿轮润滑, 接线紧固, GDT 目检, 干燥剂更换 |
| 每年 | 全频段重新扫描 (检查天线/电容有无退化), NVS cache erase + rebuild |
