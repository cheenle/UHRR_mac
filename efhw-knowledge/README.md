## 1. EFHW 基本原理

### 什么是 EFHW

EFHW (End-Fed Half-Wave) 天线本质上是**在一端馈电的半波偶极子**。相比中心馈电的偶极子，它的优势在于：

- **架设简单** — 只需一个支撑点，另一端可以直接拉到树上
- **端口/野外操作理想选择** — 一根线 + 一个小变压器盒即可工作
- **多波段运作** — 利用谐波在多个业余频段谐振

### 阻抗特性

- EFHW 天线呈现**极高阻抗**，典型值约 **2500 Ω**（远高于 50 Ω 收发机阻抗）
- 需要一个阻抗变换器（49:1 或 64:1 UNUN）匹配到 50 Ω
- 实际阻抗取决于：天线架设高度、环境、地面电导率、导线直径等变量
- 便携操作每次架设位置不同，末端馈电半波天线真实阻抗可能变化很大

### 多波段原理

以 40m 半波为例：
| 频段 | 波长倍数 | 说明 |
|------|----------|------|
| 40m (7MHz) | 1× λ/2 | 基波半波谐振 |
| 20m (14MHz) | 2× λ/2 | 两次谐波 |
| 15m (21MHz) | 3× λ/2 | 三次谐波 |
| 10m (28MHz) | 4× λ/2 | 四次谐波 |

> 注意：偶次谐波（20m, 10m）的馈电阻抗接近基波，变压器匹配较好；奇次谐波（15m）阻抗变化更大，匹配较差。

## 2. 49:1 vs 64:1 变压器设计与选择

### 匝数比选择

49:1 是 EFHW 最常用的变压比，但选择原则很简单：

> 天线的实际阻抗 ÷ 50 = 理想变压比

由于实际阻抗未知（约 2000-3000 Ω），**49:1 (√49 = 7:1 匝数比)** 提供了一个"足够好"的折中方案。

### 49:1 vs 64:1 对比

| 特性 | 49:1 | 64:1 |
|------|------|------|
| 匝数比 | 7:1 (如 2:14 或 3:21) | 8:1 (如 2:16 或 3:24) |
| 匹配阻抗 | ~2450 Ω | ~3200 Ω |
| 适合 | 通用多波段 EFHW | 低波段(80m)、高架设、40m基波短导线 |
| 优点 | 谐波匹配更平衡 | 低端效率略高 |
| 缺点 | 高波段寄生电容问题 | 高波段匹配更差 |

### 绕线方法

**传统缠绕法**：
- 初级和次级前 2 匝绞在一起提高耦合度
- 剩余匝仅通过磁芯磁通耦合
- 存在交叉绕线 (crossover turn) → 可能降低耦合效率

**Fuchs 风格分绕法** ("Outside the Box")：
- 次级单独绕在磁芯上，无交叉绕线
- 初级单独绕在次级中间
- 优点：绕组隔离，防止静电顺同轴电缆进入收发机
- 缺点：需要独立的 0.05λ 平衡地线连接在次级底部
- 建议：次级匝紧密排列以提高匝间耦合、减少漏磁

**推荐绕法**：紧密排列的 Fuchs 风格分绕法，初次级完全分开。

### W8JI 对绕线方法的重要纠正 (权威来源)

Tom W8JI 通过实际 VNA 扫描对比发现：

**"Split winding" (分绕法) 神话被揭穿：**
- 分绕法 vs 简单并绕法在 FT240-65 磁芯上的实测对比：

| 频率 | 分绕法抑制 | 并绕法抑制 |
|------|-----------|-----------|
| 15.71 MHz | -33 dB | **-47 dB (更优)** |
| 37.1 MHz | -46 dB (峰值) | -41 dB |
| 134.8 MHz | -19 dB | -4.6 dB |
| 157 MHz | -3 dB | **-11 dB (更优)** |

**结论：**
- 分绕法只是将峰值性能点**移到更高频率**
- 在 HF 频段（<30MHz），**简单并绕法反而表现更好**
- 并联电容（分布电容）**不一定有害** — 可以增加共模阻抗
- 规则：只要并联容抗 ≥ 感抗的一半，阻抗反而增加

> "Many 'improvements' we just blindly accept from reading articles" — W8JI

### 磁芯选择

| 磁芯 | 材质 | μ | Ae (mm²) | 适用频段 | 适用场景 |
|------|------|---|----------|---------|---------|
| FT240-43 | Mn-Zn Mix 43 | 850 | 161 | 40m-10m | 最通用、综合能力强 |
| FT240-31 | Mn-Zn Mix 31 | 1500 | 161 | 80m-20m | 低频段表现更好 |
| FT140-43 | Mn-Zn Mix 43 | 850 | ~70 | QRP 便携 | 体积小、功率<10W |
| **DMEGC Ni-Zn 50×25×28** | **Ni-Zn** | **800/1000** | **350** | **40m-10m (μ=800)** / **80m-10m (μ=1000)** | **🏆 高功率/80m优化** |

> **新发现 (2026-05-24)**：横店东磁 DMEGC Ni-Zn 铁氧体磁环（50×25×28mm, μ=1000, AL=2500nH, Ae=350mm²），截面积为 FT240-43 的 2.2 倍，80m 磁芯损耗仅为 FT240-43 的 43%。详细分析见下方及 `references/dmegc_nizn_toroid.md`。

核心注意：80m 频段下初级电感不足，磁芯容易进入磁饱和（magnetizing current 过大），导致发热和效率严重下降。这是 49:1 变压器在 80m 表现差的根本原因。

### DMEGC 横店东磁 Ni-Zn 磁环深度分析 (2026-05-24)

> 来源：闲鱼商品截图 + 铁氧体物理原理反推。完整工程分析见 `references/dmegc_nizn_toroid.md`。

#### 基本参数

| 参数 | 值 |
|------|-----|
| 材质 | Ni-Zn 铁氧体（横店东磁 DMEGC） |
| 尺寸 | 外径 50mm × 内径 25mm × 厚度 28mm |
| 截面积 Ae | **350 mm²**（FT240-43 的 2.2 倍） |
| 体积 | ~41.2 cm³ |
| 重量 | 202g |
| μ=800 版 AL | 1500 nH/N² |
| **μ=1000 版 AL** | **2500 nH/N²**（推荐 80m） |

#### 核心优势：为什么截面积大一倍是关键

磁芯饱和由 B = V/(4.44·f·N·Ae) 决定。Ae 大 → 相同功率下 B 低 → 磁芯损耗呈指数下降（Pv ∝ B^2.5）。

**100W 条件下各频段 B 值对比**：

| 频段 | DMEGC (Ae=350) | FT240-43 (Ae=161) | B 比值 |
|------|---------------|-------------------|--------|
| 80m (3.5MHz) | **9.2 mT** | 20 mT | 0.46× |
| 40m (7MHz) | 4.6 mT | 10 mT | 0.46× |
| 20m (14MHz) | 2.3 mT | 5 mT | 0.46× |

#### 80m 电感量与磁芯损耗

| 磁芯 | 2T 初级电感 | XL @ 3.5MHz | 总磁芯损耗 @ 80m/100W |
|------|-----------|-------------|---------------------|
| FT240-43 | 4.3 μH | 94 Ω ❌ | 5.8-17 W |
| DMEGC μ=800 | 6.0 μH | 131 Ω ❌ | 1.3-4.5 W |
| **DMEGC μ=1000** | **10.0 μH** | **220 Ω ✅** | **2.5-7.4 W 🏆** |

> **结论**：μ=1000 版在 80m 的磁芯损耗约为 FT240-43 的 43%，且电感量达标。这是目前性价比最高的 80m-10m EFHW 磁环选择之一。

#### Ni-Zn vs Mn-Zn：高频涡流的隐藏优势

| 特性 | Ni-Zn (DMEGC) | Mn-Zn (FT240-43/31) |
|------|--------------|---------------------|
| 电阻率 | **10⁵-10⁶ Ω·m** | 1-10 Ω·m |
| 高频涡流损耗 | **极低** | 随频率显著增大 |
| 10m 波段优势 | **明显更优** | — |

Ni-Zn 的电阻率是 Mn-Zn 的 10,000-100,000 倍。14-28 MHz 区间的涡流损耗远低于 FT240-43。

#### μ 频率响应（Snoek 极限估算）

| 频率 | μ=1000 版 AL 保持率 | 2T 电感 | XL |
|------|---------------------|---------|-----|
| 3.5 MHz | ~98% | 9.8 μH | 215 Ω |
| 7 MHz | ~93% | 9.3 μH | 409 Ω |
| 14 MHz | ~80% | 8.0 μH | 704 Ω |
| 28 MHz | ~50-60% | 5-6 μH | 880-1055 Ω |

> 28 MHz 时 μ 下降到 50-60%，但 2T 初级 XL 仍远超 200Ω 底线 → 不影响使用。

#### 工程参数估算（无官方 datasheet，基于同类材料物理推导）

| 参数 | 估算值 | 方法 |
|------|--------|------|
| Bs @ 25°C | 330-380 mT | 高 μ Ni-Zn 功率级标准范围 |
| Bs @ 100°C | 270-320 mT | ~15-20% 温度降幅 |
| Tc (居里温度) | 220-280°C | Ni-Zn 材料典型范围 |

| 热阻 Rth | ~3.2 K/W | 表面积+自然对流计算 |
| 100W 最大温升 (80m) | **10-22°C** | Pv × Rth |

#### 绕线方案推荐

```
材料：DMEGC Ni-Zn μ=1000 (AL=2500nH/N²)
绕法：简单并绕法（W8JI 验证推荐）
线圈：2T 初级(0.8mm漆包线) + 14T 次级(0.5mm漆包线)
阻抗比：(14/2)² = 49:1 ✓
初级电感：~10μH (2T)
内径空间：78.5mm 周长 → 14T×0.6mm=8.4mm 占用 → 充裕 ✓
CMC：FT240-43 上 7 圈 RG-58，距馈电点 3m
Counterpoise：80m → 4m 独立地线
```

### AA5TB 并联 L-C 耦合器：铁氧体变压器的替代方案 🆕

> 来源：VA3KOT 实验验证 (2026-02-04)，基于 Steve AA5TB 原始设计
> 📄 **完整分析**：[`references/aa5tb_efha_analysis.md`](references/aa5tb_efha_analysis.md) — AA5TB 原始理论体系深度解析 (MININEC 建模, 0.05λ counterpoise 法则, 台架调谐方法论, Moxon 平衡模型, 演化谱系, 工程限制)
> 🔗 原始页面：https://www.aa5tb.com/efha.html

#### 原理

传统 49:1 使用**宽带铁氧体变压器**在多个频段同时工作。AA5TB 耦合器采用**并联谐振电路**：电感和电容并联，在每个频段单独调谐到谐振。匝数比仍是 7:1（49:1 阻抗），但磁芯改为**羰基铁粉芯 (Type 2 powdered iron)**。

| 对比项 | 49:1 铁氧体变压器 | AA5TB 并联 L-C 耦合器 |
|--------|-------------------|---------------------|
| 磁芯 | Mn-Zn/Ni-Zn 铁氧体 | **羰基铁粉芯 (Type 2)** |
| 带宽 | 宽带（无需调谐） | **窄带（每频段需调谐）** |
| 磁芯损耗 | 频率上升损耗增大 | **极低（粉末铁芯高频特性极好）** |
| 制作难度 | 绕线即可，无需调谐 | 需要可变电容器调谐 |
| 80m 能力 | 差（电感不足） | 有限（VA3KOT 未成功） |
| 40-20m 能力 | 良好 | ✅ SWR<1.5:1, X=0 谐振 |
| 15-10m 能力 | 寄生电容恶化 | 可调谐到 SWR<1.5:1 |

#### VA3KOT 实验验证 (2026-02-04)

```
磁芯：羰基铁粉 Type 2 环
初级：3T（接 BNC → 50Ω）
次级：19T（接天线 + Counterpoise）
匝数比：3:19 ≈ 1:6.3（接近 7:1 = 49:1）
并联电容：16-160pF 薄膜可变电容 (polyvaricon)
并联电阻：2.7kΩ（模拟 EFHW 阻抗，实际使用时移除）
```

**实测结果**：
- 40m/30m/20m：可调谐到 **X=0（谐振）**，SWR < 1.5:1
- 17m/15m/12m/10m：可调谐到 SWR < 1.5:1，X 接近 0
- 80m：无法调谐
- 线圈是临时绕制的丑陋版本——**但效果已经很好**

> 「看起来只值一块钱——但它能用！」— VA3KOT

#### 为什么粉末铁芯可能优于铁氧体

| 特性 | 羰基铁粉 (Type 2) | Mn-Zn 铁氧体 (Mix 43) |
|------|-------------------|---------------------|
| μ | 10 | 850 |
| 频率上限 | >100 MHz | ~50 MHz |
| 磁芯损耗 @ 30MHz | **极低** | 显著 |
| 温度稳定性 | 优秀 | 一般 |
| AL (nH/N²) | ~10-20 | ~1000 |
| 需要匝数 (7:1) | 多匝（3T/21T 级别） | 少匝（2T/14T） |

> 粉末铁芯的 μ 低 85 倍 → 需要更多匝数达到相同电感 → 但高频损耗极低。这是「牺牲便利性换效率」的思路。

#### 缺点

- **每换频段需重新调谐** — 对 contesting / multi-band 不友好
- **FT8 多波段跳频时需要手动调整**
- 可变电容器体积大、不如固定变压器紧凑
- **BG1SB 场景**：如果固定在某几个 FT8 频段（如 40m/20m），调谐一次即可——实际影响不大

> 📄 VA3KOT 衍生文章：CFHW (2026-03-04) — 确认 EFHW 仅基波+二次谐波有效，多波段 EFHW 是营销。见参考链接。

## 3. 共模扼流圈 (CMC) 应用

> 详细内容见 `references/efhw_cmc.md`（ve3ips 翻译文章）

### 核心要点

**是否需要 CMC？**
- **高功率 (100W+) → 必须使用**
- QRP+ 功率 → 好习惯，但非绝对必要

### CMC 放置位置

最佳方式：在**馈电点和电台**之间放置 CMC，建议两个位置都放置（顽固 RFI 问题）：

```
[电台]──[同轴电缆]──[CMC扼流圈(FT240-43, 7圈)]──[49:1匹配变压器]──[天线导线(~半波长)]
                         ↓
                  距离馈电点 5-15 英尺
```

**重要提示**：CMC 扼流圈**不要在馈电点安装**，否则会扼流平衡线并影响驻波比。根据 ARRL 手册，扼流圈应距离馈电点至少 10 英尺（~3m）。

### 扼流圈类型推荐

| 频段 | 铁氧体 | 形式 | 匝数 |
|------|--------|------|------|
| 80-20m | Mix 31 | 环形 | FT240-31 上 5-9 圈 RG-8X |
| 40-10m | Mix 43 | 环形 | FT240-43 上 6-10 圈 RG-8X |
| 便携 | Snap-on Mix 31 | 夹式 | 沿同轴电缆间隔 3-5 个 |

## 3.5. TX/RX 分离架构：解决 EFHW 最大痛点 🆕

> 来源：PA9X (2025-08-27) — 2025-2026 社区新共识

### EFHW 的真正瓶颈：接收，不是发射

2025-2026 年社区共识转向：**EFHW 的发射性能"足够好"——真正的短板是接收端对 QRM 的极度敏感。**

**物理原因**：EFHW 末端馈电 = 电压波腹（高阻抗节点），天然对电场噪声极度敏感。城市/郊区环境中，LED灯、开关电源、PLC 等噪声源使底噪达到 S7-S9，完全淹没弱信号。

> "被听到但听不到回复" — N6CC, PA9X 共同确认的 EFHW 第一痛点

### 黄金组合：EFHW (TX) + 有源磁环 (RX)

```
[电台] ──→ T/R Switch ──→ [EFHW + 49:1]  ← 发射路径
                │
                └──→ [有源磁环天线]       ← 接收路径
                      可旋转 null 对噪声
```

**磁环天线的核心优势**：
- 噪声基底低 10-20dB（相比 EFHW 直接接收）
- 尖锐的 null（特定方向接收急剧下降）→ 旋转对准噪声源即可消除
- 直径 < 1m，便于安装在阳台/窗台

**推荐磁环方案（按价格排序）**：

| 方案 | 价格 | 适用 |
|------|------|------|
| MLA-30+ (国产) | ~¥150 | 入门试验 |
| Wellgood Loop v4 (M1GEO PCB) | ~¥300 | DIY 中等 |
| W6LVP 完整方案 | ~$300 | 一步到位 |

**T/R 切换方案**：
- 电台内置 RX ANT 接口（IC-7300, FT-DX10 等）→ 无需额外硬件
- QRO.cz RX Antenna Switch → 外部自动切换
- MFJ-1708B（停产）→ 二手可寻

### 对 BG1SB 的实际影响

如果当前在城市/郊区固定台站使用 EFHW，且底噪偏高（S7+），**投资 ¥150-300 的磁环接收天线可能是性价比最高的升级**——不需要改动发射系统。

## 4. 平衡地线 (Counterpoise) 理论与实践

> 来源: PA9X 详细分析 (2025-05-03)

### 为什么 EFHW 需要平衡地线

**常见误区**："半波 EFHW + 49:1 变压器不需要平衡地线" — **这是错误的**。

所有天线都需要返回路径：
- 偶极子 → 通过另一臂返回
- GP天线 → 通过地网返回
- EFHW → 如果不提供专用平衡地线，**同轴电缆外屏蔽层自动成为平衡地线**

### 不用专用平衡地线的后果

同轴屏蔽层充当平衡地线时：
1. 屏蔽层连接到收发机金属机壳
2. 机壳通过电源负极连到电源外壳
3. 电源外壳接地 → **整个家用电器的地线系统成为天线的一部分**
4. 洗衣机、冰箱、电脑等所有接地的金属外壳都变成了天线系统的一部分

这导致：
- **RFI（射频干扰）的#1原因** — 整个房子在辐射 RF 能量
- 高底噪（S9++）
- 辐射效率低下

### 平衡地线的理想长度

**经验公式**：0.05 × 最长工作波长

| 最长波段 | 平衡地线长度 |
|---------|------------|
| 40m | 0.05 × 40m = **2m** |
| 80m | 0.05 × 80m = **4m** |
| 160m | 0.05 × 160m = **8m** |

### 两种实现方式

**方式一：同轴线段做平衡地线**
- 在 49:1 变压器和 CMC 之间插入一段同轴电缆
- 长度 = 0.05 × 最长波长
- 这段同轴与 EFHW 天线线走向相反
- 必须自由悬挂，不能接触金属物体
- ⚠ 不推荐如果同轴继续进电台室，仍可能引起 RFI

**方式二：专用平衡地线（推荐）**
- 用一根独立导线连接在变压器外壳的接线端
- 长度 = 0.05 × 最长波长
- 沿 EFHW 反方向尽可能拉直
- 自由悬挂，不接触金属部件
- 这是最可靠的方案

## 5. 多波段 EFHW 谐波运作机制

### 谐波运作的真实情况

**偶次谐波**（20m = 2×40m, 10m = 4×40m）：
- 馈电点阻抗接近半波基波的高阻抗（~2500 Ω）
- 49:1 变压器匹配良好
- SWR 通常较低

**奇次谐波**（15m = 3×40m）：
- 馈电点阻抗为一个很高的阻抗峰值（阻抗极高）
- 49:1 变压器匹配恶化
- SWR 显著升高
- 需要天线调谐器 (ATU) 辅助

### 导线长度建议

常见多波段 EFHW 导线长度：
- **80-10m**: ~133 ft (40.5m)
- **40-10m**: ~67 ft (20.4m) — 标准配置
- **40/20/10m 短版**: 60ft + 加感线圈

BG1SB 关注点：由于主要使用 FT8，可接受一定程度的 SWR，但需要注意**功率合成器的发热管理**。

## 6. 49:1 EFHW 的真实性能与局限

> 来源: RF.Guru (ON6URE) / VNA 实测分析 (2026)

### VNA 实测揭示的问题

用 VNA 扫描一个典型的 49:1 EFHW（号称 80-10m），实测发现：

1. **80m** — 初级电感太小，磁芯严重磁化电流 → 能量损耗在发热而非辐射
2. **40m (基波)** — 工作状态最佳
3. **20m (2次谐波)** — 尚可
4. **17/15/12/10m** — 匝间寄生电容占主导，变压器变成**容性** → RF 传输差
5. **全频段** — 2匝初级无法实现宽带干净变换

**核心结论**：
> 49:1 变压器不能在 80-10m 全段同时保持高效率。低频端感性不足，高频端容性主导，"多波段 EFHW"更多是营销承诺而非干净的技术方案。

### 实用建议

- **40m/20m 专用 EFHW** 效率最高
- 如需全段覆盖，考虑：
  - 串联谐振电容补偿（加并联电容 = 频段扩展拐杖，但引入损耗）
  - **L-Network 方案**（单波段效率更高）
  - 分段切换匹配网络
- 用 VNA 实测自己的 EFHW，不要盲目相信标称频率范围

### N6CC 实测补充

- **100pF 并联电容**（常见建议）在 80m/40m CW 段**几乎没有可测量的效果**
- **变压器次级必须接地/接平衡地线** — 商业厂商声称"不需要"是错的
  - 次级冷端不接地 → 同轴屏蔽层/人体/电台外壳成为不可控的回路
  - "提供良好的直流/射频接地还能快速泄放雨雪静电积累"
- **发热测试**：FT240-43 磁芯 + 100W 60秒键控，石蜡刨花未融化（熔点 52-71°C）→ 可以接受
- **100W 在 2500Ω 阻抗下 = 500V RMS / 700V 峰值** → 需要耐压足够的绝缘，普通 600V 绝缘线勉强
- **手靠近输出端 SWR 就会变化** — 系统对近场环境的极端敏感性是 EFHW "挑剔、难调谐、射频进电台室"名声的来源

## 7. L-Network 匹配方案 (49:1 之外的选择)

> 来源: VA3KOT (John), Ham Radio Outside the Box (2025-04-30)

### L-Network 原理

L-Network 由一个串联电感(L)和一个并联电容(C)组成，替代 49:1 变压器的阻抗匹配功能。

| 对比项 | 49:1 变压器 | L-Network |
|--------|-----------|-----------|
| 带宽 | 宽带（多个频段） | **单波段** |
| 效率 | 声称较低 | 声称更高 |
| 制作 | 简单，无需计算 | 需要 L/C 计算 |
| 磁芯发热 | 存在（尤其高功率） | 无磁芯问题 |
| 漏磁 | 存在耦合不完善 | 无此问题 |
| 便携适用性 | 好（免调试） | 可能需要调试 |

### L-Network 制作

- 用可调磁芯的电感 + 陶瓷微调电容
- 计算 L/C 值后仍需微调（因为天线安装环境每次不同）
- 可变 L-match 本质上就是一个天线调谐器 (ATU)
- 可以做到 1.5:1 SWR

### VA3KOT 的建议

对于便携操作：
- 同时携带 49:1 变压器和 L-Network
- 根据不同情况选择
- 如果 CQ 没人回，先从天线效率找原因

## 8. NEC 建模与辐射特性

> 来源: Owen Duffy (owenduffy.net) — NEC-4.2 模型

### 20m 底部馈电垂直半波的 NEC 分析

- 在真实地面上，垂直 EFHW 的辐射仰角对 DX 有利
- 需要地面或地网系统才能发挥最佳性能
- 不能将 EFHW 当成魔术天线 — 它还是受物理定律约束
- 现代火腿对 EFHW 的"非凡宣传"需要审慎看待

### N6CC 的 EFHW 辐射方向图深度分析 (2026-03-13 更新)

**核心观点：EFHW 是 N6CC 在 80/60/40/30m 区域覆盖时的"最后选择"**，他更偏好偶极子。

**方向图特性 (132ft 斜拉 EFHW, 20° DX 仰角)：**
- 80m (3.55MHz) 斜拉 45° 至 90ft 支撑物 → **4 个方向图零点**：后方 -25dB 深零, 两侧 -9dB 零点
- 非"宽边辐射器" (not a broadside radiator as you might expect)
- 对特定 DX 路径需要将斜拉线**偏 45°** 对准目标方位
- **每个波段都是不同的天线** — 谐波频率上方向图零点更深、瓣更多
- 50ft 支撑物时 DX 仰角出现 **8dB 前方零点**
- 20ft 支撑物时前方零点 ≈ 10dB, 且地面损耗急剧上升

**Inverted L 配置问题：**
- 同时产生地波和天波 → 建设性/破坏性干涉 → 热点与死区（类似杨氏双缝干涉效应）
- 这些死区随电离层变化而移动，产生自致的 QSB 衰落

**NVIS 表现（6m 高水平 L 形）：**
- 80m/40m 高仰角时全向性在 1dB 以内
- 对 300km 内区域通信可用（NVIS 模式）
- **对远距离 DX 表现很差** — 但这反而是 NVIS 的优点：抑制远端 QRM

**关键结论：**
- 天线工程公理：**高增益、宽带宽、小体积 — 三选二**。EFHW 在实际中三个都不占。
- EFHW 的"多波段能力"更多是它能**承受功率**（类似假负载），而非高效辐射
- 仅适合"随机通联"（randomly working anyone, anywhere on some band）

### DIY 链式 EFHW (64:1) 用于 17/15/12/10m
> 来源: KM1NDY

5. 整个变压器做防水处理（户外使用）
6. 用 VNA 验证 SWR 曲线

## 11. 📦 A+C 双模匹配系统 — 已探索/已放弃 (deprecated)

> ⚠️ **2026-06-09 决策**: A+C 双模方案因工程复杂度太高（8 个工程问题、¥400-600 成本、户外可靠性风险）、且 V3.0 Fuchs ATU 连续伺服调谐方案已覆盖 40m-10m 全自动调谐，**不再继续推进**。本节保留作为技术探索记录。
>
> 完整深度分析：`references/efhw_ac_dual_mode.md`
> 原始方案来源：https://r3cb2mze.mule.page/

### 核心思路

传统单一 49:1 铁氧体变压器在 80m（电感不足）和 10m（容性主导）同时失效。A+C 方案将问题拆解：

| | Unit A (低频段) | Unit C (高频段) |
|------|----------------|-----------------|
| 磁芯 | 2×FT240-43 堆叠 | T200-2 粉末铁芯 |
| 拓扑 | 49:1 宽带变压器 | AA5TB 并联 LC 耦合器 |
| 覆盖 | 80m, 40m | 30m-10m (含 WARC) |
| 调谐 | 免调谐 | 可变电容逐波段调谐 |
| 效率 | ~92% | 90-95% |

两根磁芯 + DPDT 开关集成在一个 IP65 馈电盒中，一根 20m 天线导线全段覆盖。

### 关键工程发现

经过定量分析，A+C 方案的**理论方向正确**（分频段用不同磁芯材料），但当前原型有 **8 个工程问题**需要解决。按严重程度排序：

| # | 问题 | 严重度 | 改进方案 |
|---|------|--------|---------|
| 1 | **非谐振波段阻抗失配** — 20m 线在 20m/10m 是整数倍波长，端阻抗 ~50-300Ω 而非 ~2500Ω | 🔴 根本性 | Unit C 增加可切换匝比（2T/3T/4T 初级） |
| 2 | **可变电容耐压不足** — LC 谐振升压效应使 V_peak 达 1900-6300V | 🔴 安全风险 | 真空可变电容（3-15kV 额定） |
| 3 | **Counterpoise 不兼容** — 4m 在 10m 波段 = 0.38λ，已变成辐射体 | 🔴 严重 | Unit C 切换到 0.5m counterpoise |
| 4 | **可变电容户外可靠性** — 冷凝/腐蚀/触点氧化，寿命 6-24 月 | 🟡 可靠性 | 真空电容 + 干燥剂 |
| 5 | **DPDT 高压切换** — 707V peak 超出 250VAC 额定 2× | 🟡 中等 | 改为 50Ω 端切换（100V peak） |
| 6 | **设计过度** — DMEGC Ni-Zn 单磁芯可达 90%+ 效果，成本 1/5 | 🟡 设计 | 先评估单磁芯方案 |
| 7 | **Unit 间隔离** — 空闲 Unit 寄生吸收 ~2W | 🟢 轻微 | 空闲 Unit 初级短路阻尼 |
| 8 | T200-2 饱和 | 🟢 非问题 | 39 gauss vs 800 gauss 饱和值 |

### 与更简单方案的对比

| 方案 | 80m | 40m | 20m | 10m | 复杂度 | 成本 | 可靠性 |
|------|-----|-----|-----|-----|--------|------|--------|
| 单 FT240-43 2T:14T | 差 | 好 | 好 | 一般 | 极低 | ¥50 | ★★★★★ |
| **DMEGC Ni-Zn μ=1000** | **好** | **好** | **好** | **好** | **低** | **¥80** | **★★★★★** |
| A+C 双模 | 好 | 好 | 优秀* | 优秀* | 极高 | ¥400-600 | ★★ |

*A+C 的高频段效率优势约 0.2-0.4 dB，在 FT8 弱信号模式下不可感知。FT8 真正的瓶颈是 RX 噪声基底（EFHW 的 S7-S9 底噪），而非 TX 效率。*

### 推荐路线 (已废弃 → 见 V3.0 Fuchs ATU)

```
Phase 1 (完成): DMEGC Ni-Zn 单磁芯 → 全频段提升，零新增复杂度
Phase 2 (短期): TX/RX 分离 (MLA-30+ 磁环) → 解决最大痛点
Phase 3 (❌ 放弃): A+C v2.0 原型 → 被 V3.0 Fuchs ATU 自动调谐路线替代
Phase 4 (❌ 放弃): 户外可靠性验证 → V3.0 Fuchs ATU 单伺服+单电容方案可靠性远优于 A+C
```

> **当前方向**: V3.0 Fuchs ATU — ESP32-S3 驱动伺服电机连续调谐空气可变电容，纯执行机构、全自动、¥255。详见 [Section 12](#12-v30-fuchs-atu--esp32-s3-连续伺服调谐)。

### 一句话总结

> A+C 方案代表了一个有价值的探索方向，但在工程实践中被证明过于复杂（8 个问题、¥400-600 成本、户外可靠性风险）。V3.0 Fuchs ATU 以更简洁的架构（单伺服 + 单空气可变电容 + 远程 SWR）实现了相同的核心目标：全自动、覆盖 40m-10m、低成本、高可靠性。A+C 的探索经验（分频段磁芯选择、可变电容耐压、Counterpoise 兼容性）已融入 V3.0 设计决策。

## 13. V1.0/V2.0 自动谐振式 EFHW 调谐适配器 (legacy)

> **2026-06-07 | 已归档到 firmware-legacy/**

> 📄 **完整设计文档**：[`references/auto_efhw_tuner_design_full.md`](references/auto_efhw_tuner_design_full.md)
> 许可：GPL-3.0 (固件) / CERN-OHL-S 2.0 (硬件)
> 关联：[[aa5tb_efha_analysis]] · [[efhw_ac_dual_mode]] · [[deep-dive-2026-05-25]]

### 项目概述

基于 AA5TB 并联 LC 耦合器理论与 N7DDC ATU-100 开源框架，设计一台**室外架设、同轴馈电、全自动调谐**的 100W EFHW 天线适配器。

### 核心技术数据

| 参数 | 值 |
|------|-----|
| 频率范围 | 40m–10m (7.0–29.7 MHz) |
| 额定功率 | 100W PEP SSB/CW |
| 磁芯方案 | **T200-2 ×2 双叠** (羰基铁粉, μ=10) |
| 匝数比 | **2:13** → 42.25:1 阻抗比 → 匹配 ~2,112 Ω |
| 调谐方式 | **7 位 128 档纯电容扫描** (10–1,997 pF, 1pF 步进) |
| 调谐时间 | <0.2s (记忆重新调) / <2s (全扫描) |
| 电容阵列 | 10 只 **1812 / 3KV / C0G(NPO)** 高压贴片 MLCC |
| 继电器 | 7 × **Omron G5Q-14** (触点间耐压 2KV) |
| SWR 检测 | Tandem Match 定向耦合器 (FT37-43 + BAT41) |
| 供电 | **Bias-T 同轴馈电** 12V DC (无独立电源线) |
| PCB | 140×90mm 双面板, **2.5mm 物理开槽**隔离高低压 |
| 防护 | IP66 压铸铝壳 + 90V GDT 避雷 + 2.2MΩ 静电泄放 |

### 关键工程创新

1. **纯电容调谐**：废除 ATU-100 原电感扫描，7个继电器全部控制电容 → 128 档电容矩阵，调谐精度远超传统 L/C 二维扫描
2. **T200-2 100W 安全验证**：最严条件 (40m/100W) B_peak = 5.6 mT → 距 B_sat (800 mT) **143 倍裕度**，完全不可能饱和
3. **物理挖槽防爬电**：PCB 中线 2.5mm×115mm 透空槽切断 FR4 表面寄生电容的 3KV RF 高压爬电路径
4. **高压电容并联均流**：大容量位(C6/C7)强制多只并联，应对谐振环流 2.6A RMS
5. **固件热切换保护**：功率 > 15W 自动中断调谐 → 保护继电器触点不被 RF 电弧烧毁
